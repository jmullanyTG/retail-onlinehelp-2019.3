Edit Hierarchy

The edit hierarchy function will allow users to select the available levels for the model. The available levels will have targeted accuracy levels and a tolerance threshold value. Each level included will be a forecasted series, based on the supplied filter.

  • All Aggregated Together – This will allow users to establish a single rooted node. This level, if included, must always be the highest level of the group.
  • Inventory Level – The item/site combination.
  • Site Level – Lowest level of site hierarchy.
    • Site Region – Optional named site hierarchy level.
    • Site Division – Optional named site hierarchy level.
  • Item Level – Lowest level of product hierarchy (independent of site).
  • Site Groups – Configured site groups. May be hierarchal if desired.
  • Item Groups – Configured inventory groups (groups). These are distinctly different from item master groups.

Once selected, the available levels will appear in the “maintain hierarchy” wizard. Their individual target percentages and target threshold tolerances may be set.

  • Description – The name of the level.
  • Period Over Period Target Percentage – Used to grade the forecast based on MAPE. In the report, any value over the period-over-period target will appear green, indicating an acceptable forecast.
  • Period Over Period Target Threshold Percentage – The tolerance for the grading system as relates to the period-over-period target (e.g. a 5% tolerance on a target value of 70% must be at least 65% in MAPE accuracy). Any value between the period-over-period target and the low end of the threshold will appear yellow, indicating a mildly acceptable forecast. Any value below the threshold will appear red, indicating a poor forecast.
  • Year Over Year Target Percentage – The year-over-year target will aggregate the values and is expected to have a much higher tolerance percentage (as the forecast is expected to be more accurate, but less precise).
  • Year Over Year Target Threshold Percentage – The tolerance for the grading system as relates to the year-over-year target (e.g. a 5% tolerance on a target value of 70% must be at least 65% in MAPE accuracy). Any value between the year-over-year target and the low end of the threshold will appear yellow, indicating a mildly acceptable forecast. Any value below the threshold will appear red, indicating a poor forecast.
  • Level Order – The order in which the levels are applied.
