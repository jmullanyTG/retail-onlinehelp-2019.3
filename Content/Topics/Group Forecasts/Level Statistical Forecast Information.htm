Level Statistical Forecast Information

The level statistical forecast information will give detailed information related to the forecast for each of the levels displayed.

  • Name – The name of the level.
  • Description – Description of the level.
  • Forecast Event Date – The date of the forecast run.
  • Statistical Forecast Accuracy – The mean absolute percentage error (MAPE) of the forecast.
  • Statistical Method Chosen – The winning base method of the forecast tournament (for more details on the forecast tournament, see the Forecasts Module).
  • Statistical Method Chosen Parameters – For forecasting base methods with several outputs such as slope, parsimony, and smoothing constants, these will be displayed in this column.
