Creating and Editing Time Frames

Time frames are managed from the Time Frame Maintenance window, which you can get to in the following ways;

  • From the Maintenance tab, click Time Frame; or
  • From a Time Frames selection dialog box, click (Maintain Time Frames).
To create a new time frame:
  1. In the Time Frame Maintenance window, click Add in the Time Frame toolbar to open the Time Frame Editor.
  2. Follow the instructions below to edit the details of the new time frame.
 To edit a time frame:

  1. If you want to edit an existing time frame, select it in the Time Frame Maintenance grid and click Edit to open the Time Frame Editor.

  2. In the Security field, choose one of the following access options:

    • Private - only you can see and use this time frame.
    • Public - all users can see and use this time frame.

    • Shared - only the users or roles you select can see and use this time frame.

      • If you select Shared, click Edit Users next to choose the users and/or roles you want to share this time frame with.

  3. In the Time Frame Definition Details area, select a Time Frame Definition Type:

    • Static - Choose static if you want to define specific calendar dates for your time frame. For example, June 1 - June 30, 2018.
    • Dynamic - Choose dynamic if you want to define rolling days for your time frame. For example, "last 365 days" or "next 90 days".
    • Schedule - Choose schedule if you want to use a schedule from the Schedules module as your time frame.

  4. If the time frame will include more than one defined time period (such as "last three Novembers"), select Allow Multi-Definition.

    Once a time frame is created, this option cannot be changed.

  5. In the Time Frame Definition toolbar, click Add to add a time frame definition.

  6. Click in each time frame definition field to enter the values for the time frame definition.

    The fields displayed depend on the time frame definition type selected:

  7. If you selected Allow Multi-Definition above, click Add to create additional time frame definitions. Each definition appears as a new row in the editor.
  8. Click OK to save the time frame. If new, it appears as a new row in the grid of the Time Frame Maintenance window.

For an example, see Creating and Editing Time Frames.
