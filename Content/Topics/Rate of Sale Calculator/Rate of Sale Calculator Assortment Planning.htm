Rate of Sale Calculator Stand-Alone module

The Rate of Sale Calculator module helps users determine how much, in units or revenue, a particular placeholder or collection might sell per site per week. All of the functionality essential to Assortment Planning is available in this stand-alone version of the tool.

A scaled down version also exists directly in the Assortment Planning module.  The stand-alone version is accessed via Execute > Demand > Rate of Sale Scenario.

There are three main options in determining rate of sale for a placeholder/collection in planning demand with assortment planning:

  • Smart Start: this method leverages a marginal return curve to determine rates of sale for each placeholder required to fulfill a target that is set in Target Creation
  • Like Style Modeling: manually create placeholders/collections and model those choices after historical like items
  • Rate of Sale Calculator: using this tool after a user has either smart started their assortment or manually created placeholders/collections, users can refine assortment plans by using historical data to determine an appropriate rate of sale to apply to placeholders/collections within them; modeling in the Rate of Sale Calculator is extremely flexible in terms of the product, cluster set, and time dimensions that can selected to model demand with confidence

The Rate of Sale Calculator can be broken down into three main components.

  1. ROS Scenario Creation
  2. ROS Calculation
  3. ROS Application
