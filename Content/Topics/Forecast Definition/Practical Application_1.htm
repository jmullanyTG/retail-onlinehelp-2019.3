Practical Application

Any interaction with the survey rules will come in one of two places within Demand Management:

  1. The forecast viewer will display a specific trend rule if it has disqualified the method from the forecasting tournament. This functionality is described in the forecasts documentation.
  2. The “edit method definition” window within the forecast definition module.

The “edit method definition” window will show several tabs across the top of the window, each specific to a set of rules.
