Practical Application

The series rules are managed within the “edit method definition” window. A series rule may be added to a definition by clicking on the “add” button, and “delete” to remove a series rule from the definition.

Clicking on the lower “add” button will populate the series rules stages and default parameter values windows. A series rule may appear in more than one stage by clicking the “add” button multiple times.

A default value for the parameter must be chosen for any series rule which requires it.
