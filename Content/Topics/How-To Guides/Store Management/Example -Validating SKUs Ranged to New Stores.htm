Example - Viewing SKUs Ranged to New Stores

In this example, we will review the items that have been ranged to new stores.

To View Items Ranged to a New Store

  1. From the Management tab, click Inventory Manager.The Items window displays.

  2. From the Filter field, select a filter, and click OK. The Choose Fields window displays.

  3. From the Choose Fields window, select any additional fields you want to view, and click OK. The Inventory Manager Document displays. For this example, we will not select any additional fields.

  4. In the Inventory Manager Document, you can see all of the items that have been ranged for the new Calgary store.
