Editing a Workflow

  1. From the Maintenance tab, click Workflow Manager.
  2. In the Workflows list, select the workflow you want to edit.

  3. Click Edit in the Workflow toolbar.

  4. Change the following details in the Workflow dialog box and click Save:

    Field Description
    Name The workflow name appears in the Workflows list.
    Solution

    The Demand Management solution the workflow will govern. This field is not editable.
    Active Whether the workflow is "live" in the system. Consider making workflows inactive while creating or editing them to avoid unintended behavior.
    Controlled User Rights

    The set of solution-specific user rights to be controlled by this workflow. All user rights related to the selected solution are controlled by default. You can customize this set by moving rights from the Workflow Controlled list to the Available list in the Workflow User Rights dialog box.
