Saving Layouts

A layout is the combination of pivots, filters, and data fields (metrics) that appear in the Hybrid Financial Planning Grid. You can easily customize a layout by adding or removing fields, applying filters, and aggregating or disaggregating metrics. Once a layout is set up to your liking, you can save it for later use.

Saving a layout

  1. Right-click in the data area of the grid and choose Save Layout.

  2. In the Save Layout dialog box, enter or conform the following layout properties:

    • Name: The name to give the layout.
    • Description: An optional description of the layout.
    • Group: The folder in which to save the layout.
    • Default: Whether or not this is to be the default layout for all financial planning documents.
    • Private/Public/Shared: The privacy level for this layout.
      • Private: This layout is only available to you.
      • Public: This layout is available to everyone in your organization.
      • Shared: This layout is only available to certain roles and/or users. Click "..." to select who it is shared with.
  3. Click OK to save the layout.
