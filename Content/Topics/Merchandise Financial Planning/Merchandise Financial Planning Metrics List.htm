The following table describes the available Merchandise Financial Planning metrics.

Notes:

  • Blue rows indicate user-editable values, and green rows indicate "ex VAT" metrics.
  • Grayed out cells indicate metrics that are not available in a particular plan type.
  • If a metric has "Has LY?" answered with "Yes," then there is an LY version of the metric, which is named with "LY" prepended to the metric name.
  • If a metric has "Has Comparison?" answered with "Yes," then there is a comparison version of the metric, which is named with the comparison plan name appended to the metric name.
