Main Planning View

The Financial Planning grid is where you view and edit a plan's data.

Because it would be impractical to display an entire financial plan on screen at once, you can choose which part of the plan to display in the grid.

For information on customizing the grid, see:

  • Plan Parameters
  • Restoring Layouts
  • Changing Data Layouts
  • Adding, Removing, and Rearranging Fields

Cell Colors

Data cells in the grid are color-coded to indicate what type of metrics they represent and whether their values are locked.

Cell Color Description
Light blue Plannable metrics. These cells are editable. Their values represent the financial plan data.
Light purple

Comparison metrics. These cells are read-only. Their values are read directly from the comparison plan selected in the Parameters pane.
Dark Grey Actualized metrics. These cells are read-only. Their values come from the actual sales data that has been imported into Financial Planning.
Light green

Ex-VAT metrics. For plans that include VAT, these are values that exclude any amount associated with value added tax. These cells are read-only.

If the VAT % is zero for the given product/site level, ex VAT metrics will be equal to the non-ex VAT metrics (ie. Sales = Sales ex VAT, etc.)

Calculation: The generic calculation for any ex VAT metric is = non-ex VAT metric value / (1 + VAT %)
Light red

This cell value is locked. Light-red coloring is applied only to sales-related metrics. Other types of metrics do not change color when locked.

Aggregate columns

Aggregate "Total" columns are added to the grid whenever lower-level data is displayed. These columns follow the same color-coding scheme as other data cells, but use a gradient instead of a solid background. Examples:

Grid Menu

Right-click inside the data grid to see the following options.

Option Description
Best Fit Columns Resizes the column widths to best fit the column names.
Best Fit Data Resizes column widths to best fit data values.
Open Opens grid data in the selected format.
Export Exports grid data to a file in the selected format.
Export Options Sets text export options (Value or Display Text).
Restore Layout Displays plan data in the selected layout.
Restore Data Layout Changes the data fields displayed to those included in the selected data layout.
Save Layout Saves the current configuration of filters, pivots, and data fields.
Status Bar Toggles showing and hiding the status bar.
