Changing Data Layouts

The Restore Data Layout context menu option lets you change the fields used as metrics in the data area of the planning grid without changing the current filters and pivot fields.

  1. Right-click in the data area of the grid and choose Restore Data Layout > [name of desired layout]. The layout may be stored in a group (in the example below, the layout is stored in the group "My Layouts").

  2. The data fields in the grid are now replaced with the data fields from the selected layout. The pivots remain unchanged.
