Hybrid Financial Planning view functionality

The Hybrid Financial Planning view works in exactly the same way as the main planning view in terms of core functionality.  The purpose of this tab is to allow for the budgeting of growth and/or decline up and down the product and location hierarchies over time to be able to determine the sales activity in these segments of the business. 

The parameters selected in the main planning view are connected to the Hybrid Financial Planning view.  If a user makes any changes to the parameters defined in the header area, this updates for the main planning view as well as the Hybrid Financial Planning view.  Additionally, the calculation toggles and the locks that are invoked will be consistent with the selections made in the main planning view.

For more information, see Main Planning View.
