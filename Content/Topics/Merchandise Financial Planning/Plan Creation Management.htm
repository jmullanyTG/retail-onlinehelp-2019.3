Creating and Managing Financial Plans

Financial planning involves multiple types of plans:

  • Master plan - A master plan is the "single version of truth" and can only be edited through an approval and/or merge process.
  • Branch plan - A branch plan is a copy of either a master plan or another branch plan. It is a "working copy" where changes to a plan can be made. These changes can then be uploaded to the master plan through an approval and/or merge process.
  • Snapshot - Snapshots are locked-down versions of a plan that can not be edited and do not actualize.

You can create and manage these plans from the Financial Plan Maintenance window.
