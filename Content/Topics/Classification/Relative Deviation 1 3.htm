Relative Deviation 1-3

Relative deviation will set the X, Y, and Z criteria for inventory.

  • Do Not Set Relative Deviation – The task will not change the relative deviation.
  • Set Relative Deviation To Specified Value – This will assign the relative deviation of all items to a single relative deviation.
  • Calculate Relative Deviation Based On Input Values – This will use selected parameters to define the relative deviation.
    • Determine Relative Deviation Based On ___ Days – This will set the number of days of historic demand or projected demand the calculation is done. It is common to set this to 365.
    • Contribution – Contribution may be set to historic demand or forecasted demand.

The range for the relative deviation must be set for the X, Y, and Z values.
