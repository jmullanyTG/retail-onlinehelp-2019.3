Creating a Classification Task

New classification tasks can be created from scratch, or cloned from an existing task.

Use the industry best practice settings to set up the base task, which may then be cloned and modified, or disabled if another task replaces it.

To Create a Classification Task

  1. From the Execute tab, click Classification. The Classification Task screen displays.
  2. Click New. The Add Classification Task screen displays.

  1. On the General tab, enter a unique name and description for the new task.
  2. Click OK. The new task is created.

To Create a Default Task

  1. From the Classification Task screen, create a new Base task. For details, see To Create a Classification Task.
  2. Select the Default Task checkbox.
  3. Click the Default Priority tab. The Default Priority tab displays.
  1. Click the Priority 1 tab and do the following:
  2. Select the Calculate Priority Based On Input Values checkbox, and do the following:
  • In the Prioritize Based On field, enter 365.
  • From the first drop-down menu, select Inventory.
  • From the third drop-down menu, select Contribution of Forecasted Demand.
  1. In the Percentage % column, enter the priority order as follows:
    • A - 80%
    • B - 15%
    • C - 5%
    • D - 0%
    • E - 0%
    • F - 0%
  1. Click the Relative Deviation 1 tab, and do the following:
  1. In the Determine Relative Deviation Based On field, enter 365.
  2. From the drop-down menu, select Contribution of Historic Demand.
  3. In the Upper Range Value and Lower Range Value columns, enter the range as follows:

  • X = 0.5 upper range, infinity lower range

  • Y = 1 upper range, 0.5 lower range

  • Z = infinity upper range, 1 lower range

  1. Click OK. The Default task is created.
