Example - Generating Distribution Orders

In this example, you will generate distribution orders for the new item High Collar Angelina Jacket. The items in these orders will delivered to stores in the United States from distribution centers stocked in the Generating Suppler Orders example.

To Generate a Distribution Order

  1. From the Execute tab, click Order Manager. The Navigation menu displays.
  2. In the Order toolbar, click Distribution. The Items window displays.

  3. From the Filter field, select the filter with the items and sites that you wish to include in your order.

  4. From the Add Supplier Order window, do the following:

    • From the Supply Chain drop-down menu, select Redistribution.
    • From the Supply Chain drop-down menu, select Ordering: High Collar Angelina Jacket/USA.
    • Select Today's Orders.
  5. Click OK. After Demand Management calculates the recommended orders, the orders display on the Order screen. Each row represents a different recommended order, and contains basic details of the order.

  6. Double-click an order row to view the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
