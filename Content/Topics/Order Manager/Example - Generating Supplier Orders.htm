Example - Generating Supplier Orders

In this example, you will generate supplier orders for the new item High Collar Angelina Jacket. The items in these orders will delivered from the supplier to distribution centers in the United States for later distribution to stores.

Workflow

Click to open the Generating Orders workflow.

To Generate a Supply Order

  1. From the Execute tab, click Order Manager. The Navigation menu displays.
  2. In the Order toolbar, click Supplier. The Items window displays.

  3. From the Filter field, select the Ordering: High Collar Angelina Jacket/USA filter, and click OK. The Add Supplier Order: Supplier window displays.

  4. From the Add Supplier Order window, do the following:

    • From the Supply Chain drop-down menu, select Supplier - DC - Store.

    • Select Today's Orders.

  5. Click OK. After Demand Management calculates the recommended orders, the orders display on the Order screen. Each row represents a different recommended order, and contains basic details of the order.


  6. Double-click an order row to display the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
