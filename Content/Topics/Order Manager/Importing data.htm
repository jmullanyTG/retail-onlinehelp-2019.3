Importing data from spreadsheets

Data can be imported directly into the Order Manager solution from MS Excel spreadsheets.

The spreadsheet must be in the .xlsx file format.

To import data:

  1. Right-click within the grid and select the Import Excel Data menu option.
  2. Select an MS Excel spreadsheet (xlsx file format only) and click Open.

  3. Click Next to display the column mapping screen.

  4. If required, map the MS Excel columns to the metrics in the Order Manager grid using the Key Column Mapping section.

    Often the MS Excel file to be imported will have been created using column headers that do not exactly match the metrics within the Order Manager grid. The Excel Key Column displays all column headers in the MS Excel file to be imported.

    To map spreadsheet column headers to grid metrics, users can click within the adjacent field in the Grid Key Column and select the corresponding metric within the Order Manager grid.

  5. If required, map the user editable metrics in the Update Column Mapping section.

    The Update Column Mapping section displays only those metrics within the Order Manager grid that can be edited by users. The check box is checked by default indicating that data will be imported from the MS Excel spreadsheet into the corresponding user editable metric in the Order Manager grid. If the check box is unchecked, then the corresponding user editable metric in the Order Manager grid is not updated.

    To change which MS Excel column will map to a particular user editable grid metric, click in the Excel Update Column field and select a different MS Excel column from the list.

    Click Finish.

    The screen refreshes and a dialog displays indicating how many rows were updated.
