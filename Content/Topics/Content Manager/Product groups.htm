Product Groups

Product groups consisting of Item Masters, Variants or other product groups, provide the ability to assign a common advertising set of Content, Text and Images.

In Product Groups, there are several key buttons available:

Action Description

New

Provides the ability to a new product group. There are several ways to create a product group. For more information, see Create a Product Group.

Clone

Provides the ability to create a copy of the product group. The cloned product group can be modified as needed. For more information, see Clone a Product Group.

Edit

Will load the Content Details and Product Details pane with the selected product group. For more information, see Content Details and Product Group Details.

Delete

Provides the ability to delete the selected product group. For more information, see Delete a Product Group.

Filter

Provides the ability to limit the display of product groups to those containing the Items or Variants returned by the selected Filter. The black hammer icon provides the ability to modify the existing or create a new filter.
