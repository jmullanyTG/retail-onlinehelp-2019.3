Content details

When a product is in ‘Edit’ mode, the Content Details pane will load in the top right. By default, the first Content Version is selected and the Text screen is displayed. The Images tab can be selected to view Master Images. Changes to both Copy Text and Images are specific to each Content Version.

To update Content fields

  1. Complete the fields in the Content Details pane as required.
  2. Click Save.
  3. In the Select Content to Update dialog, update the fields as required.

  4. Click Ok.

    Only modified Content fields are updated to prevent unchanged fields from overwriting modified content. This is implemented at all levels (Master, Event, and Content) as well as updating the modifications to the other levels.
