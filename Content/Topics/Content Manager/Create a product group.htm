Create a Product Group

  1. Click New in the Product Groups pane and the system will display the Create New Group window.

  2. In the Create New Group window, choose a filter to use for creating a product group.

    Field Description

    Filter

    Select from one of the following radio button options to determine the way the product group will be created:

    • Use Template Filter – Provides the ability to use a customized filter to quickly create product groups. For more information, see Template Filter.
    • Use Product Filter – Provides the ability to select Item Masters or Variants to include in the group with characteristics defined in the filter. For more information, see Product Filter.
    • Use Child Product Groups – This method provides the ability to create a new product group containing existing product groups. For more information, see Child Groups.

  3. Click Ok when finished to save the new product group.

    The newly created product group will appear in the Product Groups pane. If the product group consists of child product groups, the child product groups will also be displayed below the new product group.
