Product group details

The Product Group Details pane provides the ability to modify the child products in a product group or to clone a product group.

To edit a product group, select it in the Product Groups pane and double click or click Edit to load it in the Product Group Details pane.

In Product Group Details, there are two key buttons available.

Action Description

Edit

Provides the ability to edit the contents of the product group. For more information, see Edit Child Products.

View Products

Provides the ability to view the Item Masters or Variants contained within the selected child group in Product Group Details. For more information, see View Child Products.

This button is not available if the child group contains other product groups.
