Content Manager Global Policies

Global Policies are configured to set up default settings for all Demand Management solutions. They are configured by Professional Services and should not be modified without assistance from Demand Management Professional Services or Support. Changes to some Policies may require closing and opening the Demand Management software before the changes are available so it is recommended to restart the software after any Policy change. Below is a list of global policies related to content management and the purpose of each.

To configure content management policies:

  • From the Maintenance tab, click Policies and choose Module: Content Manager in the navigation pane.

Global Policy Description
Suppress Content Image Card's Add Option If set to ‘True’, prevents you from adding a local or network Image to Content Manager by suppressing the Add button from the Images toolbar.
