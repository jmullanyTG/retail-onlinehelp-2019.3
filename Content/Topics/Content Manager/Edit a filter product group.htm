Edit a filter product group

If the product group was created using a filter, Template or product, the Edit Group window will display the filter.

Action Description

Group Name

Modify the defaulted Group Name as needed.

Product Filter

Select a different existing filter from the drop down list if needed.

Modify the Filter

If a Product Filter was selected from the drop down list, a blue hammer icon will display. To modify the selected filter click on the blue hammer icon. The black hammer icon provides the ability to modify the existing or create a new filter.

Click Ok to save the updates.
