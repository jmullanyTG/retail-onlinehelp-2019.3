Associate a view type

Each new Image added should be associated with a View Type so it will display properly in an Item Display Format. This association can be made directly in the Images screen grid by selecting the View Type drop down list and choosing a value for each Image. Each View Type can only be associated to one Image.

Action Description

Save

It is beneficial to save changes if a several Images have been newly associated to the product. If the product is in a future Promotion or on a future Event, the Content Update window will display to provide the ability to update Copy Text for the other instances as well. For more information, see Content Update window.
