Content Manager Configuration

Demand Management provides the following areas for you to configure content-related items to your specific needs.

Permissions and policies

  • Users screen: where you can configure user and role-based access to content-related features and functions.
  • Policies screen: where you can configure global policies related to content management.
  • Remove User Access dialog box: where you can configure user and role-based edit privileges for content fields in a content version.

The Content Manager Configuration window

The Content Manager Configuration window is where you can configure content fields and content versions for use in Demand Management.

Getting there

To open the Content Manager Configuration window:

  1. From the Management tab, Click Content.

  2. In the Product Content pane toolbar, click Configuration.

    The Configuration option is only available to users with the Allow Access to Content Config user right.
