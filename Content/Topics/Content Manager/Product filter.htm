Product filter

Select the second radio button option to filter the products by specifying conditions that the products to be included in the product group must meet.

Enter the product group details and click Ok.

Field Description

Group Name

Enter the product group name adhering to corporate naming standards if needed.

Product Filter

Select an existing filter from the drop down list if applicable.

Modify the Filter

If a Product Filter was selected from the drop down list, a blue hammer icon will display. To modify the selected filter click on the blue hammer icon. The black hammer icon provides the ability to modify the existing or create a new filter.
