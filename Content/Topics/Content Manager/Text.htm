Text

The Text screen will display the product’s Copy Text for the selected Content Version. Each Content Version can have its own unique set of Copy Text Fields. Copy Text configured here is the Master Copy Text for the product to be used when initializing Promotion and Event Copy Text.

Copy Text can be modified by selecting the Content Version and updating the text in the appropriate Copy Text Field.

The following buttons are available:

Action Description

Save

As Copy Text is modified, it is beneficial to save changes if there are a large number of fields being modified or if changes are being made over an extended time frame. If the product is in a future Promotion or on a future Event, the Content Update window will display to provide the ability to update Copy Text for the other instances as well. For more information, see Content Update window.

Special Character Map

Open the local computer’s Character Map window to allow copying and pasting of special characters into the Copy Text Fields.
