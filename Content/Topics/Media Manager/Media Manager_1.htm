Add media

Media from a local or network source can be added to the Media Manager repository. New media displays in the Repository grid and is available for users across the Demand Management solution suite.

  1. Click on Add to load the Explorer or Finder window.

  2. Select the media file to add and click Open.

File types supported in Media Manager

The following file types are supported in Media Manager:

  • doc
  • docx
  • xls
  • xlsx
  • ppt
  • pptx
  • pdf
  • txt
  • jpg
  • gif
  • bmp
  • png
  • tiff
