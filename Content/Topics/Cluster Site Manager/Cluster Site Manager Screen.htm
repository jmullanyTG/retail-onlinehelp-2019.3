Cluster Set Screen

From the Cluster Set screen in the Management solution, you can calculate and view specific cluster sets in both detail and chart modes, make manual modifications to site cluster assignments, and arrange multiple cluster sets in the view for easy comparison.

Cluster sets that you select to display appear as tabs with the cluster set name just below the toolbar.

To Access the Cluster Set Screen

  1. From the Management tab, click Site Clustering. The Cluster Set Management window displays.
  2. To view a cluster set document for a given cluster set, select the cluster site in the grid and click OK.The Cluster Set screen displays.

Cluster Set Management Screen Toolbar, Buttons, and Header Fields

Button Description
Manage Launches the Cluster Set Management screen.
Save

Saves changes made to the user cluster and apply user value fields since the last time the work was saved.

Undo

Undoes changes made to the user cluster and apply user value fields since the last time the work was saved.

Calculate

Calculates the cluster assignments for all stores included in the cluster set (based on the parameters and criteria specified during cluster set configuration) and populates the Cluster field with these assignments.

You can calculate multiple cluster sets at once from the Cluster Set Management screen. From there, click Allow Multiple and select the desired cluster sets and/or cluster set partitions.
Configuration

Launches the cluster set Configuration window.
Refresh

Refreshes the current view.

Header Fields

Field Description
Name

The name of the cluster set being viewed.
Description The description of the cluster set being viewed.
Cluster Type

The cluster set type. For more information on cluster set types, see Cluster Set Configuration.
Last Computed

Displays the date the cluster set was last calculated.

Tab Header Right-Click Menu Options

Option Description
Close Closes the cluster set document opened.
Close All But This Closes all open cluster set documents except the one you opened; only available when multiple documents are open.
Float Converts the document tab to a window that can be sized and positioned on the screen (or other screens, in multiple monitor configurations) .
You can also click and drag the document header to reposition it on the screen. After floating, click the document header and drag the document to the desired location. Right click the document header and click Dock as Tabbed Document to return the document to its tabbed position. You can also hover over the directional buttons of the positioning reticle that appears while dragging to see possible docked locations for the floated window.

  • New Vertical Tab Group: arranges all open documents vertically side by side. After selecting this option, right-click the document header again and select Move to Previous Tab Group to return to the last screen configuration

  • New Horizontal Tab Group: arranges all open documents horizontally stacked. After selecting this option, right-click the document header again and select Move to Previous Tab Group to return to the last screen configuration

Detail Tab

The Detail tab displays the site codes and descriptions of all stores in the cluster set and their cluster assignments.

Tab Header Fields

Field Description
Name

The site name and number.
Description The site description.
Cluster Type

The system calculated cluster assignment for the site.
User Cluster

Editable drop-down field. Displays/sets the user override cluster assignment for the store. Not required.

Apply User Value Editable check box. Indicated if the system uses User Cluster in place of Cluster as the Effective Cluster or not. Default is checked.
Effective Cluster The effective cluster assignment for the store.

Grid Options

Field Description
Select All

Selects and highlights all site records in the Site Cluster Detail grid.
Deselect All

Deselects and un-highlights all site records in the Site Cluster Detail grid.
Select from Clipboard Selects and highlights all matching site records in the Site Cluster Detail grid.
Deselect from Clipboard

Deselects and un-highlights all matching site records in the Site Cluster Detail grid.

Chart Tab

The Chart tab displays all stores in the cluster set as data points against X and Y axes for all Attributes defined in Cluster Set Configuration.

Field Description
X-Axis Attribute

Selects an attribute from those defined in Cluster Set Configuration to plot on the X (horizontal) axis of the graph.
Y-Axis Attribute Selects an attribute from those defined in Cluster Set Configuration to plot on the Y (vertical) axis of the graph.
Legend

Displays one color-coded checkbox per cluster, corresponding to the colored data points that are assigned to that cluster on the chart. If unckecked, hides the related data points.
Data Points

Colored dots and stars in the chart area: dots correspond to clusters of the same color in the legend, stars represent the centroids (K-Means types) or medoids (K-Medoids types) for the distance calculations used to assign stores to clusters. Align the cross hairs on the chart to a specific data point to display a pop-up of the store associated with that point, as well as the specific values associated with the X and Y axes at the left and bottom of the chart, respectively.

  • Show Labels: Toggles the display of data point labels for all points on the chart. Default is unchecked.
