Cluster Set Site Comparison Report

The Cluster Set Site Comparison Report compares multiple cluster sets against each other and displays how sites rank across them. This is most useful when you compare cluster sets that contain the same sites, classification structure, and clustering attributes, as well as when you compare cluster sets that contain the same sites, classification structure, and clustering attributes. For example, if you compare cluster sets created by product category with the classification A, B, and C that were calculated based on the same historical sales time frame, you can identify strengths and weaknesses of each site across your product spectrum.

Object Description
Cluster Set List Where you select the cluster sets to compare in the report.
Parameters Toggles displaying/hiding the Parameters area of the report.
Execute Generates the report and compares the cluster sets selected in the Cluster Set List.

Cluster Set Site Comparison Report

The conditional formatting in this report is determined by the Most Frequent field in the Cluster Set Site Classification Report. Any cluster set classification rankings that deviate from the Most Frequent classification(s) are highlighted in orange.

You cannot modify the data that is displayed in the report.

Column Description
Site Code The unique code for the site in the system.
Site Description The description for the site.
Classification Name The classification assigned to each site for the corresponding cluster set.

Cluster Set Site Classification Comparison Report

The conditional formatting in this report is determined by the Most Frequent field in the Cluster Set Site Classification Report. Any cluster set classification rankings that deviate from the Most Frequent classification(s) are highlighted orange.

You cannot modify the data that is displayed in the report.

Column Description
Site Code The unique code for the site in the system.
Site Description The description for the site.
Classification/ Cluster Set Count The number of times the site was ranked in the corresponding classification across all cluster sets being compared.
Most Frequent

The most frequently occurring classification(s) for the site based on the data in the Classification/ Cluster Set Count columns. If there is a tie for most frequent classification, all tying classifications are listed.
