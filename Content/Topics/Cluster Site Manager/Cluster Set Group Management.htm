Cluster Set Group Management

From the Cluster Set Group Management window, you can create, edit, and delete cluster set groups.

Action Description
New Adds a new cluster set group.
Edit Edits the selected cluster set group.
Delete

Deletes the selected cluster set group.

Deleting a group does NOT delete the cluster sets/templates within them.

Cluster Set Group Properties

Property Description
Name The name of the cluster set group.
Description The description of the cluster set group.
