Attribute Management and Assignment Overview

Attribute management has traditionally been restricted to defining core attributes in host systems and pushing these to the planning solution for use in reporting and other grouping functions.

The Attribute Management solution provides the ability to maintain new attribute families and storage of imported attribute families. In addition, Attribute Sets let users to define groups of attributes which can be assigned to certain hierarchies of product. Assignment of attribute values is a simple task for users to accomplish throughout the Demand Management system.
