Slot set pane

If a Slot Set is selected in the Event Navigation Pane, the Slot Set Pane is displayed at the bottom.

Depending on the user’s permissions and the current Event Workflow Status, the user can modify the following attributes in the Slot Set Pane:

Field

Description

Slot Set Name

The name of the selected Slot Set can be modified to adhere to the corporate naming convention. This is part of the Slot Set header.

Slot Set Template

Select from the drop down list of available Templates for the Event Type. This is part of the Slot Set header.

If the selected Slot Set contains at least one Promotion that has not been shared with the user, then the Slot Set Template drop down in the left pane is disabled.

Slot Set Status (State)

Each Slot Set in the Event automatically inherits the Status of the Event. The Slot Set Status State indicates whether the selected Slot Set is ‘In Progress’, i.e. currently working on this Status, or if it is ‘Complete’. To mark the selected Slot Set as complete, choose ‘Complete’ from the Status State drop down list and click the Event Save. For more information, see Slot Set Status Change.

If the Status State drop down is disabled, you may not have User Rights to modify it. The ability to override a Slot Set Status is a Super User privilege.

Slot Set Status Override

In some situations (e.g. to provide additional permissions or to remove permissions), it may be necessary to change one or more Event Slot Set Status to a Workflow Status that is different from the rest of the Event.

By clicking Status Override, the user can set the selected Slot Set to an Event Workflow Status that occurred previously by selecting from the drop down list, click Ok and click on Event Save. For more information, see Slot Set Status Change.

You may not see this button if you do not have User Rights to it. The ability to override a Slot Set Status is a Super User privilege.

Slot Set Description

Enter the purpose of the Slot Set.

Compositions (Read Only)

Some Events may have more than one ordering of its Slot Sets, known as Compositions, to customize the Event for specific Geographies. The Compositions grid will display the position of the selected Slot Set within each Composition of the Event.
