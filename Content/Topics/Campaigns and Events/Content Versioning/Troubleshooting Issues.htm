Troubleshooting Issues

There are several resources available to find versioning issues, and identify possible causes.

  • Event Distribution Report – displays all of the Event Compositions including Geographies and Slot Set order for each.
  • Event Pricing Report – displays the price status for every Promotion in the Event for each Geography.
  • Version Report – displays distribution versions for the selected Event Slot Sets.
  • Versioned Content Output File Export Settings – provides the ability to configure the output file settings for Events.
