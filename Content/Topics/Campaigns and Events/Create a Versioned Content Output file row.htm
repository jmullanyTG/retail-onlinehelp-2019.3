Create a Versioned Content Output file row

To add a row to the Configuration, click on New, fill in the required columns and click on Save.

Field Description

Output Column  (Optional)

Enter the column that will contain the data for the desktop publishing software plugin in the appropriate format. This may be the same as the Source Column or different if the Source Column needs to be reformatted before it can be output to desktop publishing software. If Output Column is left blank, it is given the same value as Source Column. A new column is created in the database if one does not already exist.

Data Type  (Read Only)

Displays the Output Column data type.

Export  (Optional)

Select if the field should be included in the output file. If a field should no longer be included in the file, deselect this column.

Source Column  (Required)

Select the associated Column where the necessary data is stored.

Source Type  (Required)

Select from the following source types. The selection made here will determine what columns populate the Source Column.

  • System – After initial configuration, this option should not be selected.
  • Content
  • Price Output
  • Price Output Column
  • Repository
  • Promotion Attribute
  • Event Info
  • Promotion Info

Order  (Optional)

Enter the order number for this field in the file. Unless specified otherwise, the new field is appended to the file.

Friendly Name (Required)

Enter a user friendly name that will appear in the file header and is required for the desktop publishing software plugin to work successfully. All Friendly Name values must be unique.

Comments  (Optional)

Enter the purpose of this field in the output file.
