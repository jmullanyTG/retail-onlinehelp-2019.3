Modifying a Slot Set Template

To modify a slot set template, select it from the Slot Set Template list and click Edit. The Edit Slot Set Template window opens.

For descriptions of the individual template properties, see Slot Set Template Properties.

From this window, you can perform the following actions on the slot set template:

Action

Description
Edit properties The name, event type, size, default space and item format and other properties appear on the left side of the window.
Activate or deactive the template The Active check box in the property list indicates whether or not the slot set template is enables for use in events.
Renumber slots Each slot in the slot set template is automatically assigned a number (displayed in the Slot Num property). Clicking the # button next to the Zoom slider toggles entering/exiting slot renumbering mode, where you can manually renumber slots to your liking.
Save changes The Save toolbar button saves changes made to the slot set template.
Split a slot Splits the selected slot into multiple slots. Slots cannot be split into widths or heights smaller than 48 DIUs (about 1/2 inch).
Merge slots Merges selected slots.
Equal size The Equal Size toolbar button resizes selected slots into equal heights or widths across their combined area. Slots must be adjacent to each other and aligned vertically or horizontally.
Clear all The Clear All toolbar button removes all slot configurations and resets the template to a single slot with default settings.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the slot set template.
Refresh The Refresh toolbar button refreshes the Edit Slot Set Template window with the last saved changes.
