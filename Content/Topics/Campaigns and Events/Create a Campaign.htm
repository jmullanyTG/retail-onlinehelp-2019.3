Create a Campaign

Depending on the user’s rights, they may have the ability to create Campaigns. To create a new Campaign, click the New button to display the Create New Campaign window. Enter the required fields and click Ok.

Field Description

Campaign Name (Required)

Enter the Campaign Name

Start Date (Required)

Modify the default date/time to the Campaign start date/time.

End Date (Required)

Modify the default date/time to the Campaign end date/time.

Description (Optional)

Enter a summary of the Campaign.

Tags (Optional)

If the user has the appropriate rights, a new Tag can be entered this field. Existing Tags may be selected for the Campaign by clicking on … to open the Tags window and selecting the appropriate Tags.

Budget (Optional)

Enter the allocated budget for the Campaign.

Color (Required)

Modify the default color, if desired, to help identify the Campaign in the Calendar view.

Owner Name (Read Only)

The user name of the user that is creating the Campaign.
