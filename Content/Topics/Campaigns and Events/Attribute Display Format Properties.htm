Attribute Display Format Properties

Attribute Display Format properties appear in the New Attribute Display Format and Edit Attribute Display Format windows.

Getting there

  1. From the Influence tab, click Events.
  2. In the toolbar, click Configuration.
  3. In the Event Configuration window, click the Attribute Display Formats tab.
  4. In the Attribute Display Formats tab, select an existing attribute display format and click Edit, or click New to set properties for a new attribute display format.

Modifying properties

You can view and modify the properties of an attribute display format by clicking a container in the New/Edit Attribute Display Format window. Attribute display formats can include multiple containers, so be sure the correct one is selected before modifying any fields.

  • Click Undo and Redo on the toolbar as needed to undo/redo any unsaved changes.
  • Click Save on the toolbar to save any changes made.

Property

Description

Name (Required)

 The name of the attribute display format.

Units (Required)

The unit of measure used (centimeters or inches).

Width (Required)

 The width of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the attribute display format. The minimum width allowed is 6 DIUs (about 1/32 inch).
Height (Required) The height of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the attribute display format. The minimum height allowed is 6 DIUs (about 1/32 inch).
Font (Required) The font used for text (including attribute fields) in the selected container.
Size (Required) The font size used for text (including attribute fields) in the selected container.
Color (Required) The font color used for text (including attribute fields) in the selected container.
Attribute (Optional)

To add an attribute field to a container, select the field from the Attribute list and click the down arrow. The selected field will be appended to the container. You can edit the location of the field in the Text Preview box.

Attribute fields are configured by Professional Services. Please contact Professional Services or Support for assistance with these fields.

Text Preview The text displayed in the selected container. You can add, delete, edit, or move text in this box and the changes appear automatically in the attribute display format preview.
