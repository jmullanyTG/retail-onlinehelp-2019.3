Create a Cost

To add a new cost, select New. This will create a new row in the grid for the user to modify as needed to reflect the new cost. Select Event > Save to save the new cost information and to update the Total Event Costs and Available Event Budget fields.

The user can enter the following information for each cost:

Field

Description

Cost Type (Required)

Select the Cost Type.

Estimated Amount (Required)

The projected monetary cost for the Cost Type.

Actual Amount (Required)

The actual monetary cost for the Cost Type.

Cost Level (Optional)

Select one of the following from the drop down list:

  • Event
  • Promotion
  • Slot Set

Applied To (Optional)

This drop down list refreshes after the Cost Level is selected. Select one from the drop down list depending on the selection in Cost Level:

  • Event – The system will default the current Event
  • Promotion – A list of all Promotions in the Event
  • Slot Set – A list of all Slot Sets in the Event

Supplier (Optional)

Select the drop down list from the list of Suppliers that are maintained in the Suppliers solution.

Invoice Number (Optional)

The Invoice Number associated with the cost.

Comments (Optional)

Aany additional comments for the cost.
