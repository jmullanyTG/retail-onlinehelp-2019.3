Versioned Content Output

Versioned Content can be output for use by a desktop publishing software. To configure Versioned Content Output, open an Event, select Event Workbench Tab and click on Output. This screen includes the output files configuration and the root location of the Slot Set files and Images.

Versioned Content Output file configuration

The fields to be included in the output file are configured in the ‘Versioned Content Output File Configuration’ tab of this view. They are configured by Professional Services and should not be modified without assistance from Demand Management Professional Services or Support.

There are multiple rows, identified with the Source Type ‘System’, that are required to ensure the desktop publishing software plugin works properly. They are displayed for informational purposes only and cannot be modified. The remaining rows may be configured for the plugin to populate desktop publishing software as needed or additional rows may be added.
