Text

Fields that are highlighted red in the Text tab indicate that the field is displayed in the Slot’s associated Item Display Format.

The following actions are available on the Content toolbar.

Action Description

Save

As Copy Text is modified, it is beneficial to save changes if there are a large number of fields being modified or if changes are being made over an extended time frame.

Special Character Map

Open the local computer’s Character Map window to allow copying and pasting of special characters into the Copy Text Fields.

Content Link

Allows the user to identify one Product from the Promotion to use as the source for copy and/or images.

  • Only one Product can be selected as the master source for Copy and one for Images. This could be the same Product or two different Products.
  • Any copy and/or images that were manually entered prior to linking will be overwritten.
  • The inheritance is one time only and after the information has been copied over, it may be manipulated as needed.

Hide/Show Unused Fields

Toggles hiding and displaying fields that are not used in the current promotion. 

This action is only available to event-specific content (promotions in an event).
