Create an Event (wizard)

To create a new Event from the Events solution, click Event New and the New Event Wizard window is displayed. Enter the required fields and click Ok.

Field Description

Event Name (Required)

Enter the Event Name

Campaign (Required)

Use the drop down list to modify the associated Campaign, if needed.

Campaign Start Date (Read Only)

View the Campaign start date/time.

Campaign End Date (Read Only)

View the Campaign end date/time.

Event Type (Required)

Modify the Event Type by selecting from the drop down list, if needed, for the Event.

Ad Start Date (Required)

Modify the default date/time to match the starting date that the Event will be shown or distributed to customers.

Ad End Date (Required)

Modify the default date/time to match the ending date that the Event will be shown or distributed to customers.

Sale Start Date (Required)

Modify the default date/time to match the starting date that the Event’s Promotions will be available to customers.

Sale End Date (Required)

Modify the default date/time to match the ending date that the Event’s Promotions will be available to customers.

Event Code (Optional)

Enter an Event Code if required by corporate standards.

Event Bid Status (Required)

Select from the drop down list to determine if the Event is open or closed for bids.

Bid Due Date (Required)

Modify the default date/time to match the end date that the Promotion Bids are due for the Event.

Workflow (Required)

Modify the default Event Workflow for the Event Type by selecting from the drop down list, if needed.

Tags (Optional)

If the user has the appropriate rights, a new Tag can be entered this field. Existing Tags may be selected for the Event by clicking on … to open the Tags window and selecting the appropriate Tags.

Budget (Optional)

Enter the allocated budget for the Event. During Event planning, actual costs entered by the user are aggregated and compared against this budget.

Owner Name (Read Only)

The username of the user that is creating the Event.

Description (Optional)

Enter a summary of the Event.

Clicking Next will allow review and modification of the default Slot Sets, if not appropriate. Otherwise, clicking Finish will create the Event with the default Slot Sets for the Event Type.

To add a new Slot Set, click Add. To remove a Slot Set, select the check box beside it, click Remove, then click Yes to confirm the delete. To re-order Slot Sets, select a Slot Set and click on the up or down arrows to the right of the grid to change its position in the Event.

For each Slot Set, the following may be modified as needed:

Field Description

Slot Set Name

Enter a name for the Slot Set that follows corporate naming standards.

Slot Set Template

Select a Slot Set Template from the list of Templates associated with the Event Type.

Once the Event Slot Sets have been set up appropriately, click Finish when which will cause the Event Information screen to load with the entered configuration.
