Buttons

When an Event is selected in the Event Navigation Pane, several buttons are displayed that provide the ability to make changes to the overall Event depending on the user’s permissions and rights. If a Slot Set is selected in the Event Navigation Pane, additional buttons are available. User Rights and Event Workflow permissions for the current user may affect which buttons are available.

Action

Description

Save

Allows the user to save any changes that have been made to the Event Information screen.

Clone

Provides the ability to create a copy of the current Event. All associated Promotions will be duplicated for the new Event.

Edit Header

Provides access to modify Event Header details including Campaign, Workflow, Ad Dates and Sale Dates.

Change Status

Provides access to change the Event Status based on the associated Workflow. The user may not have access to this functionality depending on the configuration Event Workflow’s Transition Rights.

Slot Set: New

New appends a new Slot Set to the Event using the default Slot Set Template defined in the Event Global Policies. The Slot Set will be appended to the first Event Composition.

The label for this button will change depending on the Event Type’s configuration.

Slot Set: Delete

Provides the ability to delete the selected Slot Set from the Event. All Promotions that were placed on the Slot Set will be removed from the Event but not deleted.

The label for this button will change depending on the Event Type’s configuration.
