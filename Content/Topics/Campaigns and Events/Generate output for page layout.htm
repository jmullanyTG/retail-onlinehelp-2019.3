Generate output for page layout

To generate output, select the Generate Output for Page Layout tab, choose the Slot Sets to output and depending on the Global Policy ‘Allowed Versioned Content Output Methods ‘, choose the desired Output Method from the drop down list. Then select Execute. The output is produced in accordance with the associated Global Policies and Versioned Content Output Configuration.

You can continue working in Demand Management while the output is being generated. You will receive a notification once output generation begins and again when it is completedy.
