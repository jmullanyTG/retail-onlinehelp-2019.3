Create a Geography

When a new Geography is created, it is possible to copy Event Promotions and Content from an existing Geography to the new one. To create a new Geography, click on Geography New. If desired, select an existing Geography to copy Promotions and Content from, click Ok, then click Event Save. The dates of the Geography will default to the Event Ad dates but may be modified. Add or remove Sites to the Geography as needed.

For more information, see Add Sites to a Geography and Remove Sites from a Geography.
