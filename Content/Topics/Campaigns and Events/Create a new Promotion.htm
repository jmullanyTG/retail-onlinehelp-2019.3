Create a new Promotion

If there is no Promotion available that is appropriate for a particular Slot, a new Promotion can be created from both the List and Layout views. This can be done by navigating to an open Slot and clicking Promotion New. Selecting this button will open the New Promotion Wizard window. Enter the required fields and click Save and Close.

Field

Description

Promotion Name (Required)

Enter the Promotion Name following your corporate standard naming, if applicable.

Start Date (Read Only)

Will default to the Event Ad Start Date or the Event Sale Start Date depending on the Global Policy setting ‘Default new Promotion with Event date range’.

End Date (Read Only)

Will default to the Event Ad End Date or the Event Sale End Date depending on the Global Policy setting ‘Default new Promotion with Event date range’.

Workflow (Required)

Modify the default Workflow if desired for the Promotion.

Price Cluster Set (Required)

Select the appropriate Site Cluster Set for the Promotion. The Site Cluster Set loaded to the Promotion will be determined based on the Site Cluster Set start date and Promotion Start Date. This user may customize the Promotion for each Site Cluster in the Promotion Detail screen.

Promotion Strategy (Optional)

Select a default Promotion Strategy for the Promotion from the drop down list. Populated with available Promotion Strategies defined in the Promotion Configuration Promotion Strategy area. When modifying the Promotion, the Promotion Strategy can changed for one or more Site Clusters.
Private, Public, Shared The default privacy option is Public. You can change the default if you wish to designate certain promotions as sensitive and restrict the ability for other users to view the promotion. Selecting Shared restricts viewing the promotion to certain users/roles.

Display Grid Fields (Optional)

The display grid on the left pane will show pricing fields that have been configured for the selected Promotion Strategy. If an editable field is available, this will allow the user to input a Promotion level value to apply across all Site Clusters. This can be customized at the Site Cluster level in the Promotion Detail screen.

Products (Optional)

If a Promotion Strategy has been specified, products may be added to the Promotion. These Products will apply to all Site Clusters in the Promotion. They may be customized per Site Cluster in the Promotion Detail screen. Depending on the type of strategy selected the manner in which products are added differs.

Once all of the required information has been entered in the Promotion, click Save and Close. After the Wizard closes, the user will either return to the Event Slot Set or will be directed to the Promotions solution Promotion Detail screen depending on the Global Policy ‘Go to Promotion Module after Promotion creation’.

If the Promotion Detail screen is displayed, further modifications to the Promotion can be made here. If the Events solution is displayed, the user can navigate to the Promotion in the Event and click on View to make changes. Content may be entered specifically for the Event by clicking on Content. For more information, See Content Editor.
