Slot Set Templates

Slot set templates define the default layouts for your slot sets. Each slot set template is a grid of one or more slots, and each slot in the template can have its own size dimensions, space, item display format, slot name, and slot number.

For convenience, you can designate the event types for which a slot set template can be used and choose whether a template should be active (available for use) or inactive (not available to users).

Slot set templates are configured in the Slot Set Templates tab of the Event Configuration window.

Previewing a slot set template

On the left side of the Slot Set Templates tab is a Slot Set Template Name list. This list contains all of your slot set templates and indicates whether or not a template is active. Select a name in the list to preview that slot set template. You can use the Zoom slider as needed to magnify the preview.

Toolbar actions

The following actions are available from the Slot Set Templates toolbar.

Action

Description
New Creates a new slot set template from scratch.
Clone Creates a new slot set template by copying the selected slot set template. The clone contains all the content and properties of the selected template. You can give it a new name as needed.
Edit Modifies the selected slot set template.
Delete Deletes the selected slot set template.
