Item Display Format Properties

Item Display Format properties appear in the New Item Display Format and Edit Item Display Format windows.

Getting there

  1. From the Influence tab, click Events.
  2. In the toolbar, click Configuration.
  3. In the Event Configuration window, click the Item Display Formats tab.
  4. In the Item Display Formats tab, select an existing item display format and click Edit, or click New to set properties for a new item display format.

Modifying properties

You can view and modify the properties of an item display format by clicking a container in the New/Edit Item Display Format window. Item display formats can include multiple containers, so be sure the correct one is selected before modifying any fields.

  • Click Undo and Redo on the toolbar as needed to undo/redo any unsaved changes.
  • Click Save on the toolbar to save any changes made.

Adding content fields, attribute display formats, price display formats, and images to an item display format

To add content fields, images, price display formats, or attribute display formats to a container, set the Container Type property to the desired type and choose the specific content fields, image repository, or formats to include in the container. You cannot mix different content types (for example, content fields and price display formats) within the same container.

The following table provides additional content-specific information.

Type Description
Attribute Display Format

When Attribute Display Format is selected as the container type:

  • An Attribute Format property is added to the properties list. This property contains all defined attribute display formats.
  • Selecting a format appends the attribute display format to the container. You can rearrange the location of the format in the Text Preview box.
  • The Font, Size, and Color properties apply to the attribute display format and any additional text entered in the Text Preview box.
Content Field

When Content Field is selected as the container type:

  • A Field property is added to the properties list. This property contains all content fields.
  • Selecting a field appends the content field to the container. You can rearrange the location of the field in the Text Preview box.
  • A Max Character property is also added to the properties list. This sets the maximum number of characters to display from the selected content field.
  • The Font, Size, and Color properties apply to the content field and any additional text entered in the Text Preview box.
Price Display Format

When Price Display Format is selected as the container type:

  • A price value is appended to the container. This is a placeholder to view the size and position of the price in the item display format before it is associated with a promotion. Once the item display format is associated with a promotion, the price display format associated with that promotion is used to display the price.
  • The Font, Size, and Color properties apply to the price and any additional text entered in the Text Preview box.
Repository Image

When Repository Image is selected as the container type:

  • A Repository Type property is added to the properties list. This property contains all image types: Feature, Inset, Logo, and Product.
  • Select the type of image you want to appear in the container. Once the item display format is associated with a promotion, the specific image associated with the promotion appears in the item display format.

Properties

Property Description
Name (Required) The name of the item display format.
Units (Required) The unit of measure used (centimeters or inches).
Width (Required) The width of the selected container. The minimum width allowed is 6 DIUs (about 1/32 inch).
Height (Required) The height of the selected container. The minimum height allowed is 6 DIUs (about 1/32 inch).
Container Type The type of content that the selected container holds. Options are:
  • Blank
  • Attribute Display Format
  • Content Field
  • Price Display Format
  • Repository Image

Selecting a non-blank option adds a new property based on the selected option (see [Field/Format/Image] below).

[Field/Format/Image]

This property's name and values depends on the Container Type selection. For example, if Attribute Display Format is selected as the container type, this property is named Attribute Format and contains a list of attribute display formats for you to add to the container. (See Adding content fields, attribute display formats, price display formats, and images to an item display format for more information.)

Selecting from this list adds the selection to the container.

Max Characters The maximum number of characters to include from the content field. Only shown when Container Type = Content Field.
Font The font used for text (including fields) in the selected container. Not shown if Container Type = Repository Image.
Size The font size used for text (including fields) in the selected container. Not shown if Container Type = Repository Image.
Color The font color used for text (including fields) in the selected container. Not shown if Container Type = Repository Image.
Text Preview The text displayed in the selected container. You can add, delete, edit, or move text in this box and the changes appear automatically in the item display format preview. Not shown if Container Type = Repository Image.
