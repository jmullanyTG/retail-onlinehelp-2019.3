Create an Event Composition

To create a new Composition, select Entry New. A new row will be added to the Compositions grid. Enter a name for the new Composition and click Save.

All available Geographies are displayed in the Composition Geographies grid. The Geography order is defined in the Geography Set configuration. Select the check box beside the Geographies to assign to the new Composition and click Save in the Composition Geographies grid.

Next, Slot Sets must be added to the Composition using the Composition Slot Sets grid. Select New in this grid to add a new Slot Set row. Add as many Slot Set rows as needed for the new Composition. For each row, enter the position of the Slot Set in the Composition and select a Slot Set from the Event. Click Save in the Composition Slot Sets grid to save the pagination of the Slot Sets for the Composition.

Slot set previews will not be displayed until Versioning has been executed.
