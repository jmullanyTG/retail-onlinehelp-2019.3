Prerequisites for the Size Curve Calculation

The following are prerequisites that must be met for the size curve calculation to run:

Stock On-hand Data by Site/Size: the size curve calculation makes use of stock on-hand data to qualify whether the sales that occurred for a given variant/period should be included when calculating a size curve for a given site. If this data is not present for the items included in the product Filter selected for the size curve, the enhanced calculation method cannot be employed and the old sales-only calculation method will be used instead

the specific check done to qualify this requirement is that tbl_inventory.InventoryLevelArchive is checked for any entries for the inventories in the Filter between the dates specified (Start Date to End Date or Last X Days). The presence of a single inventory record will qualify the check to proceed with the enhanced calculation method

Data Cleansing Thresholds: at least one of the two data cleansing threshold fields must be set to a positive value. For more information, see Data Cleansing Thresholds.
