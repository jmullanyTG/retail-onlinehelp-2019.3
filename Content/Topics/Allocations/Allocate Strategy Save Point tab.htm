Allocate Strategy Save Point tab

The Allocate Strategy Save Point tab allows users to mass apply a Save Point to all the variants in the Item Filter.

The grid in the left hand side of the Allocate Strategy Save Point tab displays save point details including creation date and who created the save point.

Field descriptions

  • Apply Save Point – Applies the save points for the selected strategies.
  • Create Save Point Before Restore – creates another Save Point saving variant settings before a restore.
  • Save Point ID – A unique identifier for the Save Point.
  • Allocate Strategy ID – A unique identifier for the strategy line. Any active product groups will also be assigned a unique allocate strategy ID.

  • Allocate Strategy Code – The identifying code for the level ID associated with the strategy. This code will be the item master group code of the hierarchal level defined in the allocation hierarchy.

  • Allocate Strategy Description – A description accompanying the code. This description will be the item master group description of the hierarchal level defined in the allocation hierarchy.
  • Product Group ID – A unique identifier for the product group.
  • Level ID – The identifying code for the level ID associated with the strategy. This code will be the item master group code of the hierarchal level defined in the allocation
