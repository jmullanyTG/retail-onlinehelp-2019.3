Workload Queue Active Statuses

The following Statuses will be seen in the Workload Queue Active list

  • Unknown - The system includes a default Status Type which is only used if the system is experiencing an issue and Demand Management Support should be contacted.
  • Preparing (a.k.a. Batching) - The system is evaluating the Items in the submitted Batch and estimated workload.
  • Coalescing - The system is marking Items in the submitted Batch to reduce processing time by removing duplicate items. Last Batch submission, with like Items, will be processed first. Prior Batches will complete the non-coalesced Items and then be placed into a Status of ‘purgatory’ until the last Batch, with ‘like’ Items, has been completed.
  • Ready - The system has completed preparation and coalescing the Batch making it ready to be executed by the Workload Queue Marshall.
  • Executing - The system is executing the submitted Batch.
  • Purgatory - The Batch is waiting for a following Batch, with like Items, to be completed.
