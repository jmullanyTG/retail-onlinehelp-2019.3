Assign a Calculation

Within the Promotion Calculations tab, click New above the Calculation Set Calculations grid to add a new Promotion Calculation to the Calculation Set. Enter the required fields and click Save.

Field Description

Calculation Order (Required)

Enter a numeric value identifying the sequence order for the new Calculation being added.

Calculation Name (Required)

Select a Calculation from the drop down list. Populated with Calculations defined in the Promotion Calculation Configuration grid.

Calculation Description (Read Only)

Defaults to the Calculation Description after changes are applied.

Active Flag (Optional)

Selected by default to identify that the Calculation added is active in the Calculation Set.
