Create a Promotion wizard

To create a new Promotion from the Promotion solution click New and the system will display the Create New Promotion wizard. The window will display fields to enter values at the top, a display grid on the left that is dynamically generated based on the selected Promotion Strategy, and a products grid on the right that is available after a Promotion Strategy is selected. Depending on the global policy ‘New Promotion View Collection Set’ a second customized tab may display in the products grid area.

Promotion Strategies available for selection

Roles and Users will only see those Promotion Strategies for which they have access. This is to ensure compliance with corporate Promotional Pricing for all advertising channels.

Creating a new promotion

  1. Complete the required fields.

  2. Selected the desired privacy option (Private, Public, Shared).

    For shared promotions, you must click the horizontal ellipse button to select the Users\Roles to share the Promotion with.

  3. Click Save and Close.

Field Description

Promotion Name (Required)

Enter the Promotion Name following your corporate standard naming, if applicable.

Start Date (Required)

Modify the default date/time to the Promotion start date/time. If the Promotion is being created from an Event, the system will use the Event Ad Start Date and the field will be restricted from modification.

End Date (Required)

Modify the default date/time to the Promotion end date/time. If the Promotion is being created from an Event, the system will use the Event Ad End Date and the field will be restricted from modification.

Workflow (Required)

Modify the default Workflow if desired for the Promotion.

Price Cluster Set (Required)

Select the appropriate Site Cluster Set for the Promotion. The Site Cluster Set loaded to the Promotion will be determined based on the Site Cluster Set start date and Promotion Start Date. This user may customize the Promotion for each Site Cluster in the Promotion Detail screen.

Promotion Strategy (Optional)

Select a default Promotion Strategy for the Promotion from the drop down list. Populated with available Promotion Strategies defined in the Promotion Configuration Promotion Strategy area. When modifying the Promotion, the Promotion Strategy can changed for one or more Site Clusters.
Private, Public, Shared The default privacy option is Public. You can change the default if you wish to designate certain promotions as sensitive and restrict the ability for other users to view the promotion. Selecting Shared restricts viewing the promotion to certain users/roles.

Display Grid fields

The display grid on the left pane will show fields that have been configured for the selected Promotion Strategy. If an editable field is available, this will allow you to input a Promotion level value to apply across all Site Clusters. This can be customized at the Site Cluster level in the Promotion Detail screen.

Products (optional)

If a Promotion Strategy has been specified, products may be added to the Promotion. These Products will apply to all Site Clusters in the Promotion. They may be customized per Site Cluster in the Promotion Detail screen. Depending on the type of strategy selected the manner in which products are added differs.

Once all of the required information has been entered in the Promotion, click Save and Close. The Promotional Detail screen displays allowing further detailed modification of the Promotion.
