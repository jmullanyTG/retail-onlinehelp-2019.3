Text

You may update the content in the configured Copy Text fields of the ‘Text’ tab as desired. The following buttons are available if needed:

Action

Description

Save

As Copy Text is modified, it is beneficial to save changes if there are a large number of fields being modified or if changes are being made over an extended time frame. If the Promotion has been placed on a future Event or the product in the Promotion has Master Content, the Content Update window will display to provide the ability to update Copy Text for the other instances as well. For more information, see Content Update window.

Special Character Map

This will launch the local computer’s Character Map window to allow copying and pasting of special characters into the Copy Text Fields.

Content Link

This allows you to identify one Product from the Promotion to use as the master source for Copy Text and/or Images.

  • Only one Product can be selected as the master source for Copy and one for Images. This could be the same Product or two different Products.
  • Any copy and/or Images that were manually entered prior to linking will be overwritten.
  • The inheritance is one time only and after the information has been copied over, it may be manipulated as needed for the Promotion.
