Add Sites to Site Cluster

To add Sites to a Site Cluster

  1. Select the Promotion Site Cluster.
  2. Click on Sites Add.
  3. The Sites selection box will display.
  4. Select the Select only some items from the current filter check box.

  5. Select the Site(s) to be added to the Site Cluster and click OK.

    The Sites added must not already be part of any other Site Cluster associated with the Promotion. Also, the Sites added must match the Currency of the Site Cluster.
