Promotion Manager User Rights

There are two sets of user rights within Promotion Management; one set not affected by the Promotion Workflow Status (i.e. global), the other affected by Status. Below is a list of global user rights and the purpose of each. These permissions can be configured by clicking Users from the Maintenance tab and using the arrow buttons to add or remove rights for a user or role.

User Right Description

Allow Access to Analysis

Provides access to the Promotion Analysis functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Access to Bids

Provides access to the Promotion Bidding functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Access to Companion Items

Provides access to the Promotion Companion Items functionality.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Access to Content

Provides access to the Content button in the Promotion Detail header that allows you to view copy and images associated with the Promotion.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Access to Promotion

Provides access to the Promotion solution.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Creation of Price Cluster

Provides access to create Site Clusters in the Promotion.

Allow Creation of Promotion

Provides access to the create Promotions.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Creation of Tags

Provide access to create new Tags in the Promotion Header.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Allow Deleting of Price Cluster

Provides access to delete Site Clusters in the Promotion.

Allow Deleting of Promotion

Provides access to delete Promotions.

Allow Editing of Content

Provides access to edit copy and images associated with the Promotion. The ‘Allow Access to Content’ right must also be granted for this to work properly.

Allow Editing of Price Cluster

Provides access to edit Site Clusters in the Promotion.

Allow Editing of Promotion

Provides access to edit Promotions.

Allow Editing of Promotion After Sale Start Date

Provide access to edit a Promotion after the Sale Start Date has occurred.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses but rather as a Super User privilege.
Allow Override of Validation Failures Provides access to mark the override check box, for a failed Promotion Validation (i.e. Information, Warning, Error), in the Promotion Validation Log. Overriding a Promotion Validation error will replace the Validation failure severity icon with the 'Failure Reviewed' icon to visually identify to users that a Validation Failure exists but has already been 'approved'.

Config - Allow Access to Promotion Config

Provides access to the Promotion Configuration window: Rounding Rules, Promotion Calculations, Promotion Validations, Promotion Pricing Details, Promotion Strategies and Export Maintenance.

It is not recommended that this permission be incorporated with Promotion Workflow Statuses

Modify of Promotion Status

Provide access to edit Promotion Status.
