Promotion details screen

Double click a Promotion in the List screen to display the Promotion’s details. On the Promotion details screen, there are several buttons and icons available to you.

Action Description

Save

Saves the current Promotion.

Clone

Creates a copy of the Promotion.

Recalculate

Available if changes have been made to the Promotion and will refresh all Promotion Calculations for all Site Clusters.

Content

Ability to update Promotion Content Copy Text and Images.

Change Status

A drop down selection allowing you to modify a Promotion’s Workflow Status or multiple Promotion Statuses at once. The options are:

  • Change Status
  • Change Multiple Statuses

Price Clusters and Sites

Manage the Promotion Site Clusters.

Site Count

Manage the Sites in the selected Promotion Site Cluster and displays the total number of Sites where this Promotion applies.

Details

Ability to view the Promotion Header and update some of the information.

Date Range

Ability to change the Promotion’s Start and End Dates.

Message

This icon is green if there’s a message available for the Promotion at the current Workflow Status.
