Create a Rounding Rule – standard or price ending

Within the relevant Rounding Rule tab, click New to create a new Rounding Rule, enter the required fields and click Save.

Field Description

Name (Required)

Enter a descriptive label to summarize the purpose of the Rounding Rule. This will be used to identify the Rounding Rule when creating Promotion Calculations.

Description (Optional)

Enter a summary of the Rounding Rule.

Active  (Optional)

Leave selected to indicate that this Rounding Rule is active and will be available when configuring Promotion Calculations.
