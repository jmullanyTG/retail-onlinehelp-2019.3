Clone a Promotion

A copy of a Promotion can be made by clicking the Clone button from the Promotion Detail screen.

The Clone Promotion dialog displays allowing you to make changes to the Promotion Header for the new Promotion.

If you select the Include Promotion check box when cloning an Event with associated Promotions, then the privacy options are carried over from the original Promotions. Also, the last Promotion Owner is retained in the new Promotion being created by the cloned Event.

Action Description

Promotion Name (Required)

Modify the default name of the new Promotion.

Date Range (Required)

Start and End Dates of the new Promotion can be modified by clicking on the … button.

Description (Optional)

Enter a summary of the new Promotion.

Priority (Optional)

Select a Priority value from the configurable drop down list.

Effect (Optional)

Select an Effect value from the configurable drop down list.

Tags (Optional)

Click Tags to associate Tags with the Promotion. If you have the appropriate rights, new Tags can be created.

Requirements (Optional)

Click on … and select one or more Requirements in the window.

Price Cluster Set (Required)

Update the Promotion Site Cluster Set if needed. The version of the Site Cluster Set loaded to the Promotion will be determined based on the Site Cluster Set start date and Promotion Start Date.

Workflow (Required)

Update the Promotion Workflow from the drop down list if needed.
Private, Public, Shared The default privacy option will be based on the original promotion being cloned. You can change the default privacy option, if you wish, for the cloned promotion.

Created By (Read Only)

Displays the name of the user that cloned the Promotion and the creation date and time.

Last Changed By (Read Only)

Displays the name of the user that last modified the Promotion and the date and time when that change was made.

Click Ok to save and load the new Promotion. This Promotion can be modified as needed.
