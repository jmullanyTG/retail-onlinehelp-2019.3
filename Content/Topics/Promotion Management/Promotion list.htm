Promotion list

The List screen provides a filtered list of Promotions with the ability to view or edit or create new, depending on your permissions. In the grid, the first check box column allows you to select multiple Promotions for deletion.

NOTE: Promotions marked as Private to other users do not display. The promotions that do display are those marked Public and those promotions shared with your user or the role(s) that your user belongs to.

Columns include:

The following columns were introduced with the v2019.2.1 Service Pack.

Column Description
Lifecycle

Indicates if the Promotion occurred in the past, future, or is currently being executed. The field displays:

  • PAST - The current system date is after the Promotion’s Sale End Date Time
  • LIVE - The current system date is between the Promotion’s Sale Start Date Time and the Sale End Date Time
  • FUTURE - The current system date is before the Promotion’s Sale Start Date Time

NOTE: This column does not display if system policy Allow Multiple Dates is set to True.
Price Cluster Set Name The Site Cluster Set Name that was used during creation. Modification of the Price Clusters and Cluster Sites after creation does not modify this value.
Price Cluster Set ID The Site Cluster Set ID that was used during creation. This is the actual Site Cluster Set, not its Snapshot ID.

Double-clicking a promotion in the grid displays a Promotion Detail screen as a separate tab.

The available Promotion Strategies displayed in the Promotion Detail Site Cluster grid are filtered based on the Promotion Strategy’s restricted User and Role access.

Note: If a previously selected Promotion Strategy is no longer valid, then the system retains the now restricted Promotion Strategy. However, once the user selects a different Promotion Strategy the restricted Promotion Strategy is no longer available.
