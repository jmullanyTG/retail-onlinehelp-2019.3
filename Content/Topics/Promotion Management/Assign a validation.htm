Assign a validation

Within the Promotion Validations tab, click New above the Validation Set Validations grid to add a new Promotion Validation to the Validation Set. Enter the required fields and click Save.

Field Description

Validation Order (Required)

Enter a numeric value identifying the sequence order for the new Validation being added.

Validation Name (Required)

Select a Validation from the drop down list. Populated with Validations defined in the Promotion Validation Configuration grid.

Validation Description (Read Only)

Defaults to the Validation Description after changes are applied.

Active Flag (Optional)

Selected by default to identify that the Validation added is active in the Validation Set.
