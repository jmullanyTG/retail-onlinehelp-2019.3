Create a Calculation Set

Within the Promotion Calculations tab, click New above the Promotion Calculation Set Configuration grid to create a new Promotion Calculation Set. Enter the required fields and click Save.

Field Description

Calculation Set Name (Required)

Enter a descriptive label to summarize the purpose of the Calculation Set. This will be used to identify the Calculation Set when configuring Promotion Strategies.

Calculation Set Description (Optional)

Enter a summary of the Calculation Set.

Clone (Optional)

Leave this check box unselected.
