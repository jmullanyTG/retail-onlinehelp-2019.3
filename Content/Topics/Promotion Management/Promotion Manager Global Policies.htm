Promotion Manager Global Policies

Global Policies are configured to set up default settings for all Demand Management solutions. They are configured by Professional Services and shouldn’t be modified without assistance form Demand Management Professional Services or Support. These Promotion Manager policies are configured by clicking Policies from the Maintenance tab.

Global Policy

Description

Allow Multiple Dates

Allow the ability to set a Promotion to run multiple times across the calendar by setting more than one Start and End Date. Should be set to ‘False’.

Allow Promotions with duplicate names

Promotions are generally uniquely identified by a combination of their names and dates. This would allow multiple Promotions with the same name and dates to exist. Should be set to ‘False’.

Allow select first group and load items when opening a Promotion

When you open an existing Promotion, if this policy is set to ‘True’ it will, by default, select the first product / product group in the Promotion.

Clone Filters On Link to Promotion

If a Filter is linked to a Promotion, a clone of the Filter will be created if this is set to ‘True’. That would allow you to make Promotion level modifications of the Filter without affecting the originating Filter.

Default CostType

Assign the default CostType to use for Promotions by its ID. If this is set to 0, it will default to CostType ‘1’.

Default Price Strategy used for new Promotions

If a default Promotion Strategy is to be defined for new Promotions, enter the Promotion Strategy ID. If the Promotion Strategy should not be defaulted, set to ‘0’.

Default PriceType

Assign the default PriceType for Promotions by its ID. If this is left blank, it will default to PriceType ‘1’.

Item Calendar View Collection Set

Assign the View Collection Set to display the Item Calendar by its ID. If this is not set to a valid ID, a blank screen would be displayed in the Item Calendar tab.

New Promotion View Collection Set

Assign the View Collection Set to display as a second tab in the New Promotion Wizard by its ID. If this is not set to a valid ID, the second tab is not displayed.

Promotion Calendar View Collection Set

Assign the View Collection Set to display the Promotion Calendar by its ID. If this is not set to a valid ID, a blank screen would be displayed in the Promotion Calendar tab.

Promotion Config View Collection Set

Assign the View Collection Set to display the Promotion Configuration by its ID. If this is not set to a valid ID, a blank screen would be displayed in the Promotion Configuration window.

Promotion Inventory Meta Data View

Assign the View Collection Set to display the Promotion Inventory in a Promotion by its ID. If this is not set to a valid ID, the default data view will be displayed.

Promotion Product Selection Level

Select ‘Item’ if the implementation will be Item Master based, or ‘Variant’ if Variants are preferred.

Recalculate Locked Promotion

If this is set to ‘True’ then even Promotions that are marked as locked can be recalculated by an overnight process.

This could result in system pricing differing from advertised pricing if the Promotion is in production.

Show Bids in Promotion View

If this is set to ‘True’, Bids functionality will be available for Promotions.

Show Companion Items in Promotion View

If this is set to ‘True’, Companion Items functionality will be available for Promotions.

Show View Collection Set for Promotion Calendar

If this is set to ‘True’, show the View Collection Set defined in the ‘Promotion Calendar View Collection Set’ policy, otherwise use the static Promotion Calendar.

Show View Collection Set for Item Calendar

 

If this is set to ‘True’, show the View Collection Set defined in the ‘Item Calendar View Collection Set’ policy or the ‘Variant Calendar View Collection Set’ policy depending on the ‘Promotion Product Selection Level’ setting, otherwise use the default Item Calendar or Variant Calendar.

Use Images from Items

This should be set to ‘True’ so Images can be copied from Items/Variants to Promotions.

Variant Calendar View Collection Set

Assign the View Collection Set to display the Variant Calendar by its ID. If this is not set to a valid ID, a blank screen would be displayed in the Variant Calendar tab.
