Promotion Header

By clicking on Details, the Promotion Header information is displayed. All required fields were provided during Promotion creation, however, depending on permissions and Workflow Status they may be modified. There are also several new fields that can be entered for the Promotion.

Field Description

Promotion Name

The name of the Promotion, initially entered on creation.

Date Range

Start and End Dates of the Promotion, initially entered on creation.

Description (New)

Enter a summary of the Promotion.

Priority (New)

Select a Priority value from the configurable drop down list.

Effect (New)

Select an Effect value from the configurable drop down list.

Tags (New)

Click Tags to associate Tags with the Promotion. If you have the appropriate rights, new Tags can be created.

Requirements (New)

Click on … and select one or more Requirements in the window.

Price Cluster Set

The Site Cluster Set used to set up the Promotion initially. This can be modified in the Promotion.

Workflow

The Promotion Workflow, initially entered on creation.

Created By

Displays the name of the user that created the Promotion and the creation date and time.

Last Changed By

Displays the name of the user that last modified the Promotion and the date and time when that change was made.
