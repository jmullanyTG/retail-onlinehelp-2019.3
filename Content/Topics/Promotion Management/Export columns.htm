Export columns

The second step to create a new Export Configuration is to define the export columns to include in the Export Columns grid. The columns configured are associated to the specific row selected in the Export Definition grid.

To create an Export Column, click New above the Export Columns grid, fill in the required columns and click Save.

Field Description

Level (Required)

Select the level of data to export from the drop down list (e.g. Group, Item, etc.)

Column (Required)

This drop down list refreshes based on the selection in the Level column. Select the column to be exported.

Include In Export (Optional)

Select this column to indicate that the column should be exported.

Description (Optional)

Enter a summary of the data to be exported.

For Promotions exports, Header columns 'ExportBatchID', 'PromotionID', 'PriceClusterID', 'PromotionName', 'PromotionStartDate' and 'PromotionEndDate' are always included.
