Create a Promotion Pricing Detail

Within the Promotion Pricing Details screen, click New above the Promotion Price Output Set grid to create a new Promotion Pricing Detail, enter the required fields and click Save.

Field Description

Name (Required)

 

Enter a descriptive label to summarize the Promotion Pricing Detail. This will be used to identify the Price Output Set when creating Promotion Strategies.

Description (Optional)

Enter a summary of the Price Output Set.
