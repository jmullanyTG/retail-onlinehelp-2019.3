Snapshot Tab

The Snapshot tab displays snapshots that have been generated for the plan that was selected in the top section of the Manage Assortment Plans screen.

Getting here

In the Assortment Plans screen, select Manage > Snapshot.

Snapshot Templates grid

A list of all snapshot templates for the current plan which the current user has access to view. The grid is used to create recurring snapshots of the current plan.

Field descriptions

Field Description
Name Name for the snapshot template.
Description Description for the snapshot template.
Remove By Days Number of days from the generation of snapshots created by a template after which the system automatically deletes them. This is used for maintenance purposes.
Snapshot Start Days Number of days backward from the date a snapshot is generated to be captured.
Snapshot End Days Number of days forward from the date a snapshot is generated to be captured.
Product Level A product hierarchy level to include in snapshots generated by the template. The available choices are based on the assigned product hierarchy for the current plan.
Site Level A site hierarchy level to include in snapshots generated by the template. Available choices are based on the assigned site hierarchy for the current plan.
Schedule The frequency and timing of snapshots generated by the template. The standard options are: Weekly Mondays, Monthly - first of month, Every 15 days. The exact timing produced by each choice is dependent on the calendar assigned to the current plan.
Snapshot Access Sets user access to snapshots generated by the template: public, private, or shared.
Snapshot Users If Snapshot Access is set to Shared, then this field sets which users to have access to snapshots generated by the template.
Template Access Sets user access to the template. The options are: public, private, or shared.
Template Users If Template Access is Shared, then this field sets which users to have access to the template.

Snapshot Templates grid toolbar functionality

Button Description
New Adds a new snapshot template line.
Delete Deletes currently selected snapshot templates from the plan.
Fields Opens the Customization dialog. Users can drag fields to and from the snapshot templates column header area and this dialog to customize the view. Custom layouts can be saved or restored by right-clicking in the Snapshot. Templates area and selecting Save Layout or Restore Layout, respectively.

Snapshots grid

Displays a list of all snapshots for the current plan, whether created manually or by a template, which the current user has access to view

Field descriptions

Field Description
Start Date Sets the start date for the snapshot window of time and must be set before End Date. This field locks after snapshot is generated.
End date Sets the end date for the snapshot window of time and must be set after Start Date. This field locks after snapshot is generated.
Remove By Date Sets the date on which the system automatically deletes the snapshot. This date must be set after the current date. Used for maintenance purposes.
Product Level A product hierarchy level to include in the snapshot. Available choices are based on the assigned product hierarchy for the current plan. This field locks after the snapshot is generated.
Site Level A site hierarchy level to include in the snapshot. Available choices are based on the assigned site hierarchy for the current plan. This field locks after the snapshot is generated.
Access Level Sets user access to the snapshot: public, private, or shared.
Access Users If Access is set to Shared, then is sets which users to have access to the snapshot.
Created From Template Name of the template the snapshot was generated from. NULL if manually generated.

Snapshots grid toolbar functionality

Button Description
New Adds a new snapshot line to the list.
Delete Deletes selected snapshots from the plan.
Fields Opens the Customization dialog. Users can drag fields to or from the snapshots column header area and this dialog to customize the view. Custom layouts can be saved or restored by right-clicking in the Snapshots area and selecting Save Layout or Restore Layout, respectively.
Save Saves changes made to snapshot templates and saves newly created templates since the last save.
Cancel Cancels changes made to snapshot templates and removes templates created since last save.
