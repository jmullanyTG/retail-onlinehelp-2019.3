Promotions Screen Parameters

The following parameters and toolbar options are available within the Promotions screen:

Toolbar options

Parameters: Shows or hides the Parameters area of the view.

Execute: Updates the Defined Promotions, Promotion Application, and Promotion Application Details sections of the view based on the selected view parameters.

Parameters

Product

A product filter for the current view.

  •   launches the Filter Conditions dialog allowing users to edit the criteria for the selected filter prior to executing it. The option also exists to create a new filter from the adjusted criteria.
  •   launches the Maintain Filter dialog, allowing the user to create or import a new filter, or update, delete, or export an existing filter.

Promotions are not tied to any group of products or product hierarchy levels. The Product selection only affects the Promotion Applications shown and products listed in the Promotion Application Details.

Given sufficient permissions, users can create promotion applications for products that aren’t included in the criteria specified by the selected product filter which causes them to disappear from the current view after clicking Execute. To view them, change the filter selection or adjust the filter’s criteria to include them again.

Site Hierarchy

The levels of the current Plan’s Site Hierarchy to include in the current view.

Similar to Product, Promotions are not tied to any Site Hierarchy levels. Site Hierarchy selection drives the Cluster Sets available for inclusion in the view. In turn, this affects the Promotion Applications shown and the clusters shown in the Promotion Application Details section.

Cluster Sets

The cluster sets to include in the current view.

The cluster sets listed include all default and additionally available cluster sets for the product level selected in the Product parameter. While cluster sets are not explicitly part of the Site Hierarchy they are directly related in that cluster sets in a given assortment plan must be contained within a single lowest level of that hierarchy. Therefore, the Cluster Set drop down will populate with the cluster sets contained beneath the selected Site Hierarchy levels.

Promotions are not tied to Cluster Sets directly. Cluster Set selection only affects the Promotion Applications shown and clusters listed in the Promotion Application Details.

Window of Time

The window of time to display in the current view. Any promotion or promotion application whose life cycle intersects with any part of the selected Window of Time will be displayed in the view.

The selected time frame must be a contiguous range of periods otherwise the system still use the window of time defined by the first and last selected periods.

Display Currency

The currency to display for all revenue based fields in the view. This parameter defaults to the Home Currency for the active Assortment Plan.

Show Completed?

Whether promotions whose end dates have passed will be displayed in the view.

Because the Window of Time selection returns all promotions whose eligibility time frame or start and end dates intersect with any part of the range selected, this is useful for quickly hiding promotions that have already passed without adjusting the Window of Time selection.
