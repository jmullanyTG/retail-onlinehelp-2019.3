Scenario Manager Scenario Details section field descriptions

This section of the Scenario Manager screen displays details about the segments users have defined for the scenarios selected in the Target Creation Scenarios section. If multiple scenarios are selected, segment data for each are stacked in the view.

Field Description
Scenario Scenario name
Target Segment Names of target segments defined for the selected scenarios
Target Filter Product filter used to define historical data for the segment
Target Filter Generated

Using Segment By Hierarchy to define segments checks this box indicating a temporary filter is used based on the product hierarchy level corresponding to the segment in place of a standard Target Filter.

For more information, see Target Segment Definitions field descriptions.
Model Filter (Optional) Product filter used in place of the Target Filter for the purpose of recommended choice count generation for the segment
Product Hierarchy Level of the plan’s product hierarchy to which placeholders generated for the segment will be assigned to
Cluster Sets Cluster sets used to define the site component of the segment. Limits the historical data returned when calculating scenarios in Target Creation.
Comparison Time Frame The time frame used to gather historical data for the segment. LY indicates the system uses the direct last year equivalent periods for the Window of Time defined for the scenario.
Collection Option

The configuration of collections to which placeholders generated for the segment will belong to.

For more information, see Creating Scenarios with Target Creation.
New Variants Maximum number of new choices defined for the clusters assigned to the segment
Total Variants Maximum number of new choices and carryover choices defined for the clusters assigned to the segment
Active Variant Threshold

Percentage (10.00%). The % of deseasonalized average sales removed during step 3 of data cleansing.

For information on data cleansing, see Rationalization .
