Variants View (only available in Collection mode)

For users who prefer to work in Collection mode, the Variants supporting view is provided to still give access to the individual placeholders that exist within each collection. Variants view automatically updates to show the placeholders within the selected collection.

Header area functionality

The header area functionality in Variants view is essentially the same as those in the Main Planning view. The exception is the ability to change the data filter parameters since the view is linked to the selected collection in the Main Planning view.

Planning area and metrics

The planning area in Variants view functions the same as the Main Planning view in Placeholder Mode including: all metrics, the ability to add/remove columns via the column chooser in View Options, and the ability to save and restore layouts.

Style bank

Users are able to use the style bank to assign variants to placeholders in Variants view in the exact same manner as in Placeholder mode of the Main Planning View.
