Cluster Set Validation Tab

The Cluster Set Validation tab displays the validation status of cluster sets for the current plan. This view has standard Pivot Grid functionality, including field customization, saved layouts, and the ability to export or open analysis in a variety of popular formats including Microsoft Excel and Adobe PDF.

Cluster sites must be configured for use with Assortment Planning (their Module property must include Assortment Planning). See, Cluster Set Configuration Screen. For more information, refer to the Cluster Site Manager User Guide.

Parameters

Window of Time: The window of time to display cluster set validations for in the Assortment Cluster Set Validation section.

Toolbar buttons

Parameters: Toggles between displaying or hiding the parameters.

Execute: Populates the Assortment Cluster Set Validation section with validation statuses for the selected Window of Time.
