Assortment Plan Defaults By Time tab

This tab is used to make adjustments to Minimum Presentation defaults across time by product hierarchy level to account for seasonal fluctuations in store inventory needs; values entered in this tab will override the Minimum Presentation value set in the Assortment Plan Defaults tab for the corresponding level of the product hierarchy for the periods in which overrides are made.

This tab features standard Pivot Grid functionality, allowing the user to manipulate the field layout by dragging and dropping fields in the Filter, Row, Column, and Data areas of the grid.

Aggregation

Time based fields (Year, Half, Quarter, Month, Week) can be removed from the grid to allow entry of Minimum Presentation for larger time frames at once. When viewing Minimum Presentation at aggregate time levels, the system will display the average across all periods beneath the level being viewed.

Entering Minimum Presentation at aggregate time levels

When entering Minimum Presentation at aggregate time levels, the system will push down the entered value to all periods beneath.

Tab parameters

Window of Time: Sets the time range to display in the Assortment Plan Defaults By Time tab.

Toolbar options

Graph: activates the standard Pivot Grid graph, which graphs any data series selected by the user

Fields: launches the Pivot Grid Fields Customization dialog with two tabs, Simple and Advanced. Each tab contains a search bar in which you can enter text to more quickly locate the fields you wish to manipulate. The fields shown in the view will be limited to only those containing the text you have entered.

Simple: fields can be dragged and dropped between Filter, Row, Column, and Data areas of the Pivot Grid to manipulate the current layout

Dragging fields to the None Area hides them in the layout.

Advanced: Manipulate the layout here by selecting the desired location of each field from the Pivot Area drop downs next to each; within each area, an Index can be specified to define the order in which the fields display.

Custom Layouts: changes made to the Assortment Plan Defaults By Time layout can be saved for future use. After making changes and clicking Ok in the Fields Customization dialog, right click anywhere in the Pivot Grid and choose Save Layout.

Field Description
Name Enter a name for a new layout, or select an existing layout using the drop down menu
Description Enter a description for the layout; not required
Default Check this to make the layout you are saving the default layout that loads when you launch the Manage Assortment Defaults dialog
Private/Public/Shared

Whether the current layout is only available for the current user, for all users, or for specified users only.

When selecting Shared, click the ellipsis (...) button to select the desired users

OK: saves the current layout with the settings you have specified

Cancel: cancels saving the layout without applying any changes to Description, Default, or Privacy settings.

Restore a custom layout by right-clicking anywhere in the Pivot Grid, choosing Restore Layout, and selecting the layout to restore.
