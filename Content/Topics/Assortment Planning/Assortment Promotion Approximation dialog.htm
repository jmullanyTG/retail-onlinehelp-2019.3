Assortment Promotion Approximation dialog

The Assortment Promotions Approximation dialog is displayed by clicking in the Approximation field in the Promotion Application section in the Promotions view.

Toolbar buttons

Button Description
Export Exports and saves the Pivot Grid in many common formats including Microsoft Excel and Adobe PDF
Open Opens the Pivot Grid in many common formats including Microsoft Excel and Adobe PDF – does not require saving prior to viewing in the desired format
Graph Graphs the selected data in the view
Fields Allows the user to customize the field layout for the view. Custom layouts can be saved and restored by right-clicking in the Pivot Grid and selecting Save Layout or Restore Layout, respectively
Save Saves changes made to the Include field by site since the last Save
Cancel Undoes changes made to the Include field by site since the last Save

Grid columns

Column Description
Promotion Application ID The unique system identifier for the promotion application.
Seasonal Index Decimal (1.01) The aggregate seasonal index for all placeholder/clusters within the period based on their assigned seasonality curves.
ROS Currency ($10.01) displays the average sales revenue per site prior to the promotion application for the period for all placeholder/clusters included in the promotion application.
ROS Units The average sales units per site prior to the promotion application for the period for all placeholder/clusters included in the promotion application.
Sales Currency ($10.01). Total sales revenue prior to the promotion application for the period for all placeholder/clusters included in the promotion application.
Sales Units The total sales units prior to the promotion application for the period for all placeholder/clusters included in the promotion application.
Promo Index Decimal (1.01). The promotional index for each period for the promotion application being approximated.
Effective Index Decimal (1.01). The index applied to period rate of sale to calculate period sales. Derived by multiplying the seasonal and promo indices for each period then dividing by the average result of all periods from the minimum Phase In date to the maximum Phase Out date for all placeholder/clusters included in the application.
AUT Currency ($10.01). The average unit ticket price for the period for all placeholder/clusters included in the promotion application
Adjusted Sales Currency ($10.01). The total sales revenue after the promotion application for the period for all placeholder/clusters included in the promotion application.
Adjusted Sales Units The total sales units after the promotion application for the period for all placeholder/clusters included in the promotion application.
AUR Currency ($10.01). The average unit ticket price after the promotion application for the period for all placeholder/clusters included in the promotion application.
Sales Difference Currency ($10.01). The difference between the sales revenue for the period before and after the promotion is applied. The calcuation is Adjusted Sales – Sales.
Sales Difference Units The difference between the sales units for the period before and after the promotion is applied.

The number of periods shown before and after the periods the promotion is to be active is controlled by the Promotion Planning Time Periods system policy (Maintenance > Policies > Module: Assortment). This system policy will not cause the approximation to include periods before the minimum Phase In Date or after the maximum Phase Out Date of all placeholders included in the promotion application, or of the single placeholder being approximated, when used in the Promotion Application Detail section.

The calculation is Adjusted Sales Units – Sales Units.
Applied By The user name of the person who created the promotion application.
Apply Date The date the promotion application was created.
