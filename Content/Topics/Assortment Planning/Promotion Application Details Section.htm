Promotion Application Detail Section

This section displays placeholder/cluster details for the application selected in the Promotion Application section. Users can make placeholder/cluster level changes to the application Start and End Dates, Final Indices, and Hold Sales parameters.

Toolbar buttons

Button Description
Fields Opens the Customization dialog in which users drag fields to or from the tab’s header area and this dialog to customize the view
Save Saves all changes to promotion application details made since the last save
Cancel Undoes changes to promotion application details made since the last save

Grid columns

Column Description
Product Eligibility The Product Filter used to define the placeholders to include in the selected promotion application
PH/PLM Phase Dates A time frame representing the Phase In and Phase Out dates of each placeholder included in the selected promotion application
PH/PLM Code Either the placeholder codes or PLM codes (for placeholders with variants assigned) for all placeholders included in the selected promotion application
PH/PLM Description Either the placeholder descriptions or PLM descriptions (for placeholders with variants assigned) for all placeholders included in the selected promotion application
Approximation Clicking in this field launches the Assortment Promotion Approximation dialog.

All fields are the same as at the promotion application level, but the data displayed is only for the specific placeholder on the line the button was clicked on. Aggregations are only of clusters assigned to that specific placeholder rather than all placeholder/clusters included in the entire application.

Cluster sections

Field Description
Start The start date (as a period) for promotion application for the specific placeholder/cluster; defaults to the Start Date for the promotion application. This date cannot be set prior to the Start Date for the promotion application.
End The end date (as a period) for promotion application for the specific placeholder/cluster; defaults to the End Date for the promotion application. This date cannot be set after the End Date for the promotion application.
Hold Sales Whether the promotion application has the ability to increase total sales of the specific placeholder/cluster. When:
  • Checked: (Enabled) the total sales of the placeholder/cluster will remain the same as before the promotion was applied, and only magnitude of sales in each period will be adjusted.
  • Unchecked: The total sales of the specific placeholder/cluster can increase above the magnitude prior to promotion application; this is effected through recalculation of the placeholder/cluster rate of sale.
Included? Whether the specific placeholder/cluster is included in the promotion application once it is made active.
Final Index The Promotional Index for the specific placeholder/cluster. Defaults to the Final Index of the selected promotion application. Cluster sections for placeholders that are not currently assorted to the corresponding cluster are grayed out and locked for editing.
