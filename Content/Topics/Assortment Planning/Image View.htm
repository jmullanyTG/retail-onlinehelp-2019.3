Image View

Use Image view to get a visual summary of your assortment plan, either by cluster or by period. Here, you can get a sense for how your assortment transforms over time. You can also make quick visual edits dynamically to see if the removal or certain items helps or hinders your visual goal.

 

  • Image view is not available in Collection mode.
  • The filter parameters in this view are linked with those in the Main Planning view; when parameters change and are executed there, this view will update and refresh accordingly. Note that it is possible to change the filter criteria within Image view to. For example, focus on a smaller window of time.

  • Due to the large data size associated with image files, selecting too much data to show in the view can cause the system to become sluggish and take a long time to refresh.

    The system policy Image View Maximum number of SQL Data Rows (Maintenance > Policies > Module: Assortment) can be used to limit the amount of data that can be brought back in Image View and help mitigate such issues. The view will raise an error if the data selection exceeds the value set in this policy.

Image mode vs. Image View

Image mode is designed especially for single-monitor users who want a bigger workspace for images than Image View provides. Image View is great for multi-screen users, as the view can be detached from the Demand Management screen and moved to a second monitor.

For more information on Image mode, see Image Mode.

Click the View Options button in the top left of the view to displays a drop down menu with options that allow you to configure the information that displays. The following menu options are available:

Making Cells Inactive

Users can make placeholders temporarily inactive by double-clicking any cell to make visual assessments, If a placeholder is present in more than one cell, either by cluster or period, then all cells related to the placeholder are made inactive automatically.

Making placeholders temporarily inactive has no effect on the Active status of placeholders in the Main Planning view.

Configuring and maintaining images

To configure and maintain images:

  1. Right click on an image in the grid and select Edit to launch the Image Gallery dialog for the selected placeholder (not available in Collection mode).

  2. If required, click New to add a new cell.
  3. Select a cell and click Carousel.
  4. In the cell, right-click and select Edit.
  5. Scroll down the list of images displayed in the Image Maintenance screen and select an image.
  6. Click Save.
  7. Click Save in the Image Gallery.
  8. Click Execute to refresh the view.

The Image Gallery screen

  • Classification: Displays the classification of each image.

    The first image assigned to a placeholder is always the primary by default, but more images can be assigned with user defined classifications.

  • Primary?: Whether an image is the primary image to display whenever images are shown for the placeholder.

  • Multiples?: Whether multiple images are allowed for the given classification.

    The Primary and Multiples parameters for variant images can be set in Management > Variant Manager > Image Maintenance.
