Excluding Unwanted Messages

You can create filters that exclude unwanted messages from appearing in the Broker Dispatch Viewer, removing clutter from your message display.

Filtered messages are not deleted, just suppressed from view in the Broker Dispatch Viewer screen.

Message Lock Exclude Filter Management dialog box

Exclude filters are managed from the Message Log Exclude Filter Management dialog box.

  • To open the Message Log Exclude Filter Management dialog box, click Exclude Filter in the Broker Dispatch Viewer toolbar.

Creating an exclude filter

  1. In the Message Log Exclude Filter Management dialog box, click New.

  2. Enter the following values for the new filter:

    • Exclude Filter Description - A brief name or description for the filter.
    • Exclude Filter Type - Select what type of value the filter will exclude. Options are:
      • Message Regular Expression: Select to filter based on the content of the message.
      • Affable Message Regular Expression:
      • Message Event Log Entry Type: Select if you want to filter messages by their severity (Information, Warning, or Error).
      • Message Type: Select if you want to filter messages of specific types (Dispatch, Progress, Application, Database Dispatch Execute Action Parallel, API, or Unknown).
    • Exclude Filter Value(s) Enter or select the values you want to exclude from view. Options are based on the Exclude Filter Type selected.
      • For expression filter types, you can type or paste the value directly into the field.
      • For all other types, select one of the preset values.
    • Security - Choose whether to make the filter private (available only to you), public (available to all Demand Management roles and users), or shared (available only to selected users and/or roles).
      • If you select Shared, click Change to select the users and/or roles to share the filter with.

    • Is Enabled - Select this option to enable the filter. When unchecked, the filter is disabled.

  3. Click OK to save the filter

Editing an exclude filter

  1. In the Message Log Exclude Filter Management dialog box, select a filter and click Edit.
  2. Edit the filter values as desired. See Creating an exclude filter above for descriptions of filter fields.
  3. Click OK to save your changes.
