Schedule Data Mapping

The schedule data mapping section will determine how the data returned in the schedule data selection method is used.

  • ID – Identification number for the schedule data object.
  • Name – Name for the schedule data object.
  • Description – A description of the schedule data object.
  • Start – The start datetime for the schedule data object.
  • End – The end datetime for the schedule data object.
  • Color – Defines the color of the schedule data object.
  • Resource ID – Defines the link between the schedule tree and the schedule data.
  • All Day Flag – If the schedule data object consumes the entire day, this flag may be checked.
  • Double Click Action – Clicking on “manage action” allows users to define an action associated with a mouse double-click.
