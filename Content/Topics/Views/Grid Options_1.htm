Grid Options

  • Allow Column Moving – If selected, users may rearrange columns within the grid.
  • Allow Column Resizing – If selected, users may change the width of the columns.
  • Allow Filter – If selected, users may add a filter to the column.
  • Allow Group – If selected, determines whether end-users can drag the column's header to the group panel to group data against the column's values.
  • Allow Quick Hide Column – If selected, users may temporarily hide a column within the grid by dragging it to the data area.
  • Allow Row Sizing – If selected, users may change the height of the rows.
  • Allow Sort – If selected, users may sort the column data by ascending or descending order.
  • Multi-Select – If selected, users may highlight and select multiple rows within the grid (by using the shift key).
  • Show Auto Filter Row – Will display the auto filter row.
  • Show Group Panel – Will show the group panel if groups are allowed.
  • Show Horizontal Lines – Will display the horizontal grid lines.
  • Show Vertical Lines – Will display the vertical grid lines.
  • Enable Appearance Even Row – If enabled, will add a subtle color highlight to every other row within the grid to make it easier to read.
