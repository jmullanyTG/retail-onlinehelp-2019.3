Linking a Widget to a View in SQL

While it is typical to link a widget to a view in the user interface using the method described in the practical application section, they may also be linked with a SQL script as follows (example is for merchandise financial planning):

  -- LINK VIEW AND WIDGET

IF @iViewID IS NOT NULL

AND @iWidgetID IS NOT NULL

AND NOT EXISTS ( SELECT 1

FROM dbo.tbl_VW_ViewWidget

WHERE ViewID = @iViewID

AND WidgetID = @iWidgetID )

BEGIN

INSERT INTO dbo.tbl_VW_ViewWidget

( ViewID, WidgetID, WidgetOrder )

VALUES ( @iViewID, @iWidgetID, 0 )

END
