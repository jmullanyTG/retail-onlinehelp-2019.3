Layout Options

  • View Caption – When using the form view, the caption will appear as a header to the form. This field is technically a format string and may use the features inherent in their creation to return desired data.
  • Show View Caption – If selected and in cases where there is a view caption, it will be displayed.
  • Card Caption Format String – The format string for the card caption.  Format string entries will vary depending on the data type. Examples include:
    • P2 – A percentage with two decimal places.
    • F3 – A fixed point decimal with three decimal places.
    • C – A currency.
    • D6 – A decimal with six decimal places.
  • Show Card Caption – If selected and in cases where there is a card caption, it will be displayed.
  • Show Header Panel – Will display the header panel.
  • Show Expand Button – Will display the “expand button”.
  • Allow Hot Track Fields – When using the form view, will add additional highlight functions to mouse-over.
  • Allow Sort – Allows sort options for the entire grid. Will override the column level settings.
  • Allow Filter – Allows filter options for the entire grid. Will override the column level settings
  • Single Card Stretch to Width – In layout mode with a single card, this selection will automatically stretch the width of the card to fit.
  • Single Card Stretch to Height – In layout mode with a single card, this selection will automatically stretch the height of the card to fit.
  • Multi Card Stretch to Width – In layout mode with multiple cards, this selection will automatically stretch the width of the cards to fit.
  • Multi Card Stretch to Height – In layout mode with multiple cards, this selection will automatically stretch the height of the cards to fit.
  • Layout View Mode – If the view mode is set to “layout”, this will determine how the layout is displayed.
  • Use Advanced Runtime Customization – When using the form view, will enable the customizable functions so that users may edit and fine tune the layout.
  • Show Save Load Layout Buttons – Will display the save and load layout buttons.
  • Allow Customization – Will allow users to customize the form at runtime.
