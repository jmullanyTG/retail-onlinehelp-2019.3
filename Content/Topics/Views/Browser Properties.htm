Browser Properties

The browser properties section will determine how the browser itself behaves.

  • URL – In instances where the URL is hard coded and not intended to be returned as a parameter, this would be the location of the URL.
  • Allow URL Edit – Will make the URL field available to the viewer so that they may edit it.
  • Allow Context Menu – If enabled, will allow the “right click” functions within Internet Explorer (i.e. view source and save).
  • Web Browser Shortcuts Enabled – If selected, will enable familiar shortcuts within Internet Explorer (i.e. F5 to refresh).
  • Script Errors Suppressed – Will suppress any script errors encountered by the browser.
  • Show URL Edit – If selected, the URL bar will be displayed.
  • Show Back Button – Will show the back button.
  • Show Forward Button – Will show the forward button.
  • Show Refresh Button – Will show the refresh button.
  • Allow Navigation – Restricting the “allow navigation” will prevent users from navigating to another page after its initial page has been loaded within Internet Explorer.
  • Allow Web Browser Drop – If selected, will allow users to use the “drag-and-drop” features of some web sites, which permit users to upload files by dropping them within the browser window.
