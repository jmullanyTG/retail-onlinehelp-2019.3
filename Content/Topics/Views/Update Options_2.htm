Update Options

  • Refresh Data on Add – Will refresh the grid if data is added which may be returned.
  • Refresh Data on Update – Will refresh the grid if data is updated which may be returned.
  • Refresh Data on Delete – Will refresh the grid if data is deleted which may no longer be returned.
  • Confirmation Message on Successful Update – Will present a confirmation message upon successfully updating data.
  • Confirmation Message – The message which will be presented (default: “Changes Saved”).
  • Allow Confirmation Message Before Add Operation – If selected, will present a confirmation before data is added.
  • Confirmation Message Before Add Operation – The message which will be presented (Default: “Allow Add Operation?”).
  • Allow Confirmation Message Before Delete Operation – If selected, will present a confirmation before data is deleted.
  • Confirmation Message Before Delete Operation – The message which will be presented (Default: “Allow Delete Operation?”).
  • Refresh Widget Siblings – Must be checked in order for dependent widgets within the view require data updates.
