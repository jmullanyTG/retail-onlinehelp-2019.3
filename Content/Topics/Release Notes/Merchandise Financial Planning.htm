Merchandise Financial Planning

The following enhancements are now available in the Financial Planning solution.

Viewing Assortment Plan data within the Merchandise Financial Planning module DMDEV-62904DMDEV-59322

Two new comparison types let users review Assortment Plan data within the Merchandise Financial Planning module. Users can now easily reference Assortment Plan data while working on Financial Plans without having to go back and forth between two screens. This helps ensure that Financial Plans and Assortment Plans align for accurate forecasting and buying. The new comparison types are:

  • vs. Assortment Plan - compares the plan's metrics against a particular Assortment Plan.
  • vs. Assortment Plan Snapshot - compare the plan's metrics against a particular snapshot of an Assortment Plan.
