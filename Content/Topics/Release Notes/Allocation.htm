Setting the prepack threshold and prepack run mode strategy lineDMDEV-62912 and DMDEV-62911

Two new columns in the Allocation Strategy grid in the Allocation module allow users to set the strategy line prepack threshold and prepack run mode. The columns are labeled:

  • Maximum Prepack Overage Allowed
  • Prepack Run Mode
