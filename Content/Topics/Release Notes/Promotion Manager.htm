Promotion Manager

The following Promotion-related enhancements are now available.

The Promotion List screen’s grid now includes three new columns to better filter Promotions by Site Cluster Set as well by whether the Promotion occurred in the past, occurs in the future or is currently being executed. The new columns are:

Column Description
Lifecycle Indicates if the Promotion occurred in the past, occurs in the future, or is currently being executed. The field displays:
  • PAST - ‘Now’ is after the Promotion’s Sale End Date Time
  • LIVE - ‘Now’ is between the Promotion’s Sale Start Date Time and the Sale End Date Time
  • FUTURE - ‘Now’ is before the Promotion’s Sale Start Date Time

NOTE: This column does not display if system policy Allow Multiple Dates is set to True.
Price Cluster Set Name The Site Cluster Set Name that was used during creation. Modification of the Price Clusters and Cluster Sites after creation does not modify this value.
Price Cluster Set ID The Site Cluster Set ID that was used during creation. This is the actual Site Cluster Set, not its Snapshot ID.
