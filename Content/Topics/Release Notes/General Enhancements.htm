General Enhancements

project = DMDEV AND fixVersion = v2019.3 AND resolution = Done AND type not in (Initiative, Task, Sub-task, Epic)

The following enhancements affect multiple solutions.

Selecting or editing multiple rows in the Break by Size pane DMDEV-64788

In Order Manager documents generated from the Assortment Planning module, the Break by Size pane now allows the user to simultaneously modify multiple rows.

Displaying the Cluster Set Name and Cluster Description DMDEV-64787

Two new optional columns in the Order Document grid allow you to display the Cluster Set Name and Cluster Description. The new columns are hidden by default.
