The following enhancements are now available in the Inventory Manager solution.

New status bar on the Stock Holding Projection screen

A new status bar displays additional metadata based on the row selection or cells within a row. You can highlight certain cells within a row or a whole row and view the sum, average, count, minimum, and maximum values in the status bar at the bottom of the grid. To display the status bar, right-click in the grid and select the Status Bar menu option.
