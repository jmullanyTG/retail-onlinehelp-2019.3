Engine

Demand Management version 2019.3 introduces a new solution called Engine Template that provides a central source of managing engine templates so that users can easily create and update templates in a single place.

There are two default task detail templates that can be used by the Engine to run items requested by the user without requiring any input from the user about the configuration. These templates are configured by editing a Replenish Task dialog or a Forecast Task dialog and saving the updated configuration into the template.

The purpose of these templates is to allow processes that should not require user input to be able to create engine tasks with appropriate predefined settings.

Engine Template is located under the Management tab.
