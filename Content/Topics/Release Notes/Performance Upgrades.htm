Performance Upgrades

Performance upgrades have been made throughout Demand Management, including the following areas:

  • Allocation - Read XML performance optimization for pre-built XML DMDEV-57214
  • API - Improved Batch Session table performance
  • Assortment - Sales Plan performance improvements
  • Database - Performance improved by using a persisted table and index in the database
  • Database - Performance improved when inserting, updating, and deleting inventory pricing by modifying the data compression
  • Events - Performance improvements when generating Event output for third party page layout solutions DMDEV-63180
  • Events - Resolved memory leak in the Event Slot Set (aka. Page) layout control DMDEV-64707

  • Events - Performance improvements when loading Event Pages

  • Events - Performance improvement loading the Event Available Promotion panels

  • Financial Planning - Optimized the MFP maintenance process DMDEV-60168

  • Inventory Policy - Improved performance when using inventory dynamic policies DMDEV-41279

  • Notifications - Improved the time to display the Notification screen

  • Price Manager - Performance Improvements to the home screen

  • Price Manager - Performance improvements to Price Action creation
  • Price Manager - Performance improvements to the Price Action Validation log
  • Profiles - Improved Cache refresh and Apply-step performance
  • Promotions - Performance improvement to Promotion creationDMDEV-60186
  • Promotions - Performance improvement to Promotion cloning DMDEV-62916

  • Workload Queue - Queued Item performance has been improved
