Site Clustering

For all Cluster Set Types, additional menu functionality lets users quickly select and deselect Site records for mass modification.

  • Select All selects and highlights all Site records within the Site Cluster Detail grid
  • Deselect All deselects and un-highlights all Site records within the Site Cluster Detail grid
  • Select from Clipboard selects and highlights all Site records that have matching VARCHAR strings from the system clipboard, for any visible column in the Site Cluster Detail grid
  • Deselect from Clipboard deselects and un-highlights all Site records that have matching VARCHAR strings from the system clipboard, for any visible column in the Site Cluster Detail grid
