Installation Notes

Refer to the Demand Management Installation Guide for detailed instructions and information related to upgrading from prior versions of Demand Management (JustEnough).

Minimum requirements

Software

.Net Framework 4.7.1 and Dot Net Core 2.1.3 are required on the Application Server and on Demand Management clients. These can be installed with the services or downloaded and installed from Microsoft.

Application Server

Recommended operating system:

  • Windows Server 2016 (or newer)

Database Server

The Demand Management database is created on a SQL Server 2017 environment, which is the earliest version of SQL recommended for a new implementation.

For a more comprehensive list, see Microsoft Lifecycle Policy.

Operating system (Application and Database Server):

  • Windows Server 2016 (or newer)

SQL Server:

  • SQL Server 2017

Workstation

Supported operating systems:

  • Windows 10
  • Windows 8.1
