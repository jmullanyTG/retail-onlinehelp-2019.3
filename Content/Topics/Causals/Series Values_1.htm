Series Values

How the values are listed and used by the engines will depend on the type of causal.

  1. For normal (additive) causals, the values will typically relate to an increase or decrease in the overall volume of units. Negative values may be entered if applicable.

  2. For manual (multiplicative) causals, an entered value of 10 = 10% increase while an entered value of -20 = 20% decrease.

  3. Event causals are used to flag a period which is associated with an event. As such, the series values are irrelevant beyond being used as a unique identifier; any number or letter may be used. Any events which span multiple periods should have a different unique identifier per period if the desired effect is to completely remove the event’s effect. If the desired effect is the use the event across multiple future periods, any events with the same unique identifier will average the effect for the future periods.

  4. Informational causals will have series values similar to normal causals.
