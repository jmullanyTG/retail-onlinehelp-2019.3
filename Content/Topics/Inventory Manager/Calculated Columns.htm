Adding Calculated columns

Calculated columns may be added to any Inventory Manager view to aggregate, average, or otherwise calculate any appropriate data.

When creating a calculated column, users will need to name the new column and choose the data type.

Format strings may be used to determine how the data is displayed (it may also be left blank). Format string entries will vary depending on the data type. Examples include:

  • P2 – A percentage with two decimal places
  • F3 – A fixed point decimal with three decimal places
  • C – A currency
  • D6 – A decimal with six decimal places

The expression editor may be used to build expressions in a similar manner to the function bar in spreadsheet applications.

Clicking on “fields” will bring up the list of columns in the current view. Advanced functions or mathematical operators may be used to build the column output.
