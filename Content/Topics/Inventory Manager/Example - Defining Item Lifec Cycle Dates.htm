Example - Defining Item Life-Cycle Dates

This example describes the setting up of a retail channel phase-in date for all style/color variants of a new item.

Setup

The following objects are used in this example.

  • A filter that includes the new item and the old item: New Items (Creation <=14 Days).

  • An Inventory Manager view that includes Site Group 3 Description and phase in/phase out fields: Life Cycle Management.

To Define an Item Life-Cycle Date

  1. From the Management tab, click Inventory Manager. The Items window displays.

  2. From the Filter drop-down menu, select the filter that contains the items you want to manage, and click OK. The Choose Fields window displays.

    If the Items window does not automatically dispplay, click Inventory Item Management on the toolbar.

  3. From the Choose Fields window, select the Life Cycle Management view, and click OK. The Inventory Manager Document displays.

  4. In the Inventory Manager Document, filter the Product and Channel columns to identify the new item (in all its variants) and the desired channel (retail).

    1. In the Product Description column filter, enter: High Collar Angelina Jacket.
    2. In the Site Group 3 Description column filter, enter Retail.

  5. To set the same Phase In Date and/or Phase Out Date for all style/color variants of the new item, do the following:

    1. Right-click in an item row and select Select All.
    2. Right-click in an item's Phase In Date field and select Multiple Changes.

    3. Select a date roughly 30 days in the future, and click Save.

    1. Right-click in an item's Phase Out Date field and select Multiple Changes.
    2. Select the Phase Out Date, and click Save. The Inventory Manager Document displays the new phase-in and phase-out dates.
  6. Click Save on the toolbar to save the new phase-in and phase-out dates.
