Defining Item Lifecycle Dates

You can define phase-in and phase-out dates for each of your items. These dates must match the dates an item will first and last appear on shelves in your store locations. Demand Management uses these dates to optimize demand forecasts, purchase orders, and other related features for the item.

To Define the Lifecyle Date for an Item

  1. From the Management tab, click Inventory Manager. The Items window displays.

  2. From the Filter drop-down menu, select the filter that contains the items you want to manage, and click OK. The Choose Fields window displays.

    If the Items dialog box does not automatically appear, click Inventory Item Management on the toolbar.

  3. From the Choose Fields window, select or create a view that includes the following fields:

    • Inventory Planning: Phase In Date
    • Inventory Planning: Phase Out Date

    • Fields you can use to identify the items you want to manage. For example, Demand Planning: Site Group 3 Description.

  4. Click OK. The Inventory Manager Document displays.
  5. In the Inventory Manager Document window, filter the columns to locate the items and/or channels in need of phase-in and/or phase-out dates.

  6. To set dates for a single item, click in its Phase In Date and Phase Out Date fields and select the desired dates.

  7. To set the same Phase In Date and/or Phase Out Date for multiple items, do the following:

    1. Filter columns until only items with the same desired dates appear in the grid.
    2. Right-click in an item row and select Select All.
    3. Right-click in an item's Phase In Date field and select Multiple Changes.

    4. Select the desired phase-in date for the selected items.

    1. Right-click in an item's Phase Out Date field and select Multiple Changes.
    2. Select the desired phase-out date for the selected items.
  8. When all phase-in and phase-out dates have been entered, click Save on the toolbar to save the dates to the items.
