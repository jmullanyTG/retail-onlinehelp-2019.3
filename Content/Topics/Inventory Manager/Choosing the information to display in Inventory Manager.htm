Selecting items, view, and fields in Inventory Manager

Choosing the items and information to display in Inventory Manager is a simple task. The configurable views can be used to create custom reports for any information within Demand Management.

Views are created and opened in separate tabs within the Inventory Manager module. Upon opening a new view, users will be prompted to select a filter of inventories.

Choosing items to review:

  1. Navigate to Management > Inventory Manager.

  2. Select the items to view by clicking the drop down arrow in the Filter field and selecting items.

  3. Click Ok.
  4. In the Choose Fields dialog window:
    1. Select a view from the Views pane.

    2. Select fields in the Fields in View pane.

      The search box near the top of the Choose Fields window can be used to filter the fields and find specific measures.

      Fields are organized by category and may be included or excluded by category, or individually using the arrows.

       

  5. Click Ok.

Choosing different items to review

To review different items:

  1. Click the Inventory Manager Document button on the top left of the screen

  2. Select the items to view by clicking the drop down arrow in the Filter field and selecting items.

  3. Click Ok.
  4. In the Choose Fields dialog window:
    1. Select a view from the Views pane.

    2. Select fields in the Fields in View pane.

  5. Click Ok.

The Choose column button within the Inventory Manager view may be used at any time to edit the included fields. Fields may be added or removed as necessary.
