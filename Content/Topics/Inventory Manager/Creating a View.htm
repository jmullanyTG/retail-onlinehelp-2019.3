Creating a View

In order to reference metadata, it must either be contained in a specific view, or referenced within an existing table. An initialization procedure (INIT) should also be created to populate this newly created view. A primary key must be included within the view in order for functions within tbl_Common_Filter_MetaViewData to work.

Example

Example for uvw_Site_SiteCapacity:

SET ANSI_NULLS ON

GO

 

SET QUOTED_IDENTIFIER ON

GO

 

 

CREATE VIEW [dbo].[uvw_Site_SiteCapacity]

AS

SELECT

si.SiteID

,SUM(c.SellingCapacity) AS TotalSellingCapacity

,SUM(c.StockroomCapacity) AS TotalStockroomCapacity

FROM tbl_Site si

LEFT OUTER JOIN tbl_Site_Capacity c

ON c.SiteID = si.SiteID

GROUP BY si.SiteID;

 

GO

Once created and populated, the view may be referenced within tbl_FilterType.
