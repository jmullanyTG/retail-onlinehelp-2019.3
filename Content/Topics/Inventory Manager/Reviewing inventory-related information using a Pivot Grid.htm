Reviewing inventory-related information using a Pivot Grid

The grid on the Inventory Manager screen can be converted to a Pivot Grid by clicking the Pivot button.

The pivot configuration tool can be used to configure the Pivot Grid. Pivot Grid reports may be built by dragging the fields into the appropriate pivot area.

Clicking the Graph button displays a graph at the bottom of the screen. A variety of chart types are available via the drop down menu in the Chart Type field.
