Stock Holding Projection

The stock holding projection tab will display the familiar “sawtooth” diagram which is key throughout the inventory planning process. Users can configure the colors of the various available series within the graph selecting a color palette which is most appealing and accessible for them.

The ability to configure a series’ color requires user right Allow Stock Projection Series Configuration.

An Advanced Calculate button displays when the item is a Replenishment item. This button requires User Right Allow Stock Projection Calculation Using Advanced Calc button.

A status bar displays additional metadata based on the row selection or cells within a row. You can highlight certain cells within a row or a whole row and view the sum, average, count, minimum, and maximum values in the status bar at the bottom of the grid. To display the status bar, right-click in the grid and select the Status Bar menu option.

Allocation vs Replenishment projections

The Stock Holding Projection tab displays allocation- or replenishment-related projections depending on whether the selected item is managed by the Allocations module or the

Demand Forecasting/Inventory Planning/Replenishment suite ("Replenishment", for short).

If you have both the Allocations module and the Replenishment suite enabled in your instance of Demand Management, the Allocation Strategy Selection field in the Allocation Strategy screen sets whether an item is managed by Allocations or Replenishment. You can add the Execution Process column to your Inventory Manager view to see which process manages each item. Its read-only values are "Allocations engine" or "Replenishment engine".
