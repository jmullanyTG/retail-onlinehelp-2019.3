Forecast Viewer

The Forecast Viewer tab will display identical information to that which may be found in the Forecasts module. Additional information may be found within the Forecasts module help documentation. Users can configure the colors of the various available series within the graph selecting a color palette which is most appealing and accessible for them.

The ability to configure a series’ color requires user right Allow Series Configuration.

Allocation vs Replenishment forecasts

The Forecast Viewer tab displays allocation- or replenishment-related forecast data depending on whether the selected item is managed by the Allocations module or the

Demand Forecasting/Inventory Planning/Replenishment suite ("Replenishment", for short).

If you have both the Allocations module and the Replenishment suite enabled in your instance of Demand Management, the Allocation Strategy Selection field in the Allocation Strategy screen sets whether an item is managed by Allocations or Replenishment. You can add the Execution Process column to your Inventory Manager view to see which process manages each item. Its read-only values are "Allocations engine" or "Replenishment engine".

Inventory view and level forecast view status bar

A status bar is supported in both inventory and level forecast views which lets you display additional metadata based on the row selection or cells within a row. You can highlight certain cells within a row or a whole row and view the sum, average, count, minimum, and maximum values in the status bar at the bottom of the grid.

To display the status bar, right-click in the grid and select the Status Bar menu option.
