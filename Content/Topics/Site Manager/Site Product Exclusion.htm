Site Product Exclusion Set

The hierarchy used to populate the product dimension in the Site Product Exclusion widget is determined by the Site.ItemMaster.ExcludeFilterItemMasterHierarchyID system policy filter.

Site product exclusion helps you to manage inventory restrictions at a site level. To create restrictions, you can add item masters filters to the exclusion list or select segment of the product hierarchy. The product hierarchy used by the site product exclusion is determined by the system policy settings.

Field Description
Site Code

The numeric site code of the selected site.
Site Description Description of the selected site.
Exclude Filter Type

Determines the method for product exclusion. The eligibility does not overlap.

  • ItemMaster Hierarchy Level: if selected, the Exclude Levels field becomes active so you can select segments of the product hierarchy for exclusion.
  • ItemMaster Filter Master: if selected, the Filter Master ID field becomes active so you can select products for exclusion based on an item master filter.
Field Master Master ID

If selected, users may select products for exclusion based on an item master filter.
Exclude Levels

If selected, users may select segments of the product hierarchy for exclusion.

Only one product exclusion can be set per site at a time. You must either create an item master filter to fulfill the requirements, or set the exclusion at an applicable level of the hierarchy.

ItemMaster Hierarchy Level

When you select ItemMaster Hierarchy Level from the Exclude Filter Type, a tree wizard displays the selected hierarchy as set by the system policy.

Levels to be excluded must be selected.

ItemMaster FilterMaster

When you select ItemMaster Filter Master from the Exclude Filter Type, you must select a filter from the filter chooser to be applied.

Product Exclusion Set

From the Site Product Exclusion Set tab, you can create reusable product exclusion sets based on combinations of item filters and product hierarchy levels. After created, a product exclusion set can be quickly assigned to any site in the Site Manager Grid.

  • Product Exclusion Exists: indicates if a product exclusion set has been applied to the selected site.
  • Product Exclusion Set: specifies the product exclusion set applied to the site.
