Example - Confirming Open Dates for New Sites

In this example, we will find new sites imported from the ERP and confirm that their opening dates are accurately scheduled . Site opening dates help determine when item orders need to be placed for the new stores.

To View Open Dates for a New Store

  1. From the Management tab, click Site Manager. The Items window displays.

  2. From the Filter drop-down menu, select a filter. The Site Manager document displays two sites (Calgary and Nashville).

  3. In the local Columns toolbar, click Choose and use the arrows to select the fields you wish to display in the document. In this example, we selected Site Type, Site Status, and Site Group.

  4. Click OK. The Site Manager screen displays the required information.
  5. In the document, double-click one of the sites (Calgary or Nashville) to display its details. If the Details pane is hidden, click Details on the toolbar to display it.
  6. On the Site Status Over Time tab, find the Closed entry. If the Status End Date field is blank, select the day before the store is scheduled to open.
  7. On the Entry toolbar, click Save.
  8. If there is not already an Open status entry for the site, click New in the Entry toolbar to add a new status, and enter the following information in the respective fields:
  • Site Status: Open
  • Site Status Attribute: Open
  • Status Start Date: The date of the store opening
  1. From the Entry toolbar, click Save.

  2. From the Site Manager grid, review the values in the Days to First Open, Days to Status Change, Next Site Status Date, Next Status Description, and other status-related fields to make sure thy are correct.
