Site Status

Site status refers to when a site will be open (such as a new store opening) or closed (such as for remodeling or permanent closure). The system uses site status to generate accurate forecasts and order recommendations, so it is important to keep sites updated with status changes as they are planned.

The Site Status tab contains a grid listing when the status of a selected site is scheduled to change. Each upcoming status change is listed as a row in the grid.

Site Status Fields

To add fields to the grid, click Fields in the Entry toolbar.

Field Description
Site Code

The numeric site code of the selected site.
Site ID The site identification number.
Site Status

Identify if the site is open or closed during this period.
Site Status Attribute

Describes the reason for the status change and specifies if the orders are accepted at the site during this period.

Examples:

  • Remodel (accepting orders)
  • Expansion (no orders)
  • Permanently closed
Notes Optional user-entered notes.
Site Status Temporal ID The site status temporal identification number.
Status Start Date The date on which this status is to go into effect.
Status End Date The date on which this status is to end.

To Add a New Status Entry for a Site

  1. Open a Site Manager document containing the desired site.
  2. Select the site in the document grid. Make sure that the Details pane is displayed.

  3. From the Site Status tab, locate the Entry toolbar, and click New. A new row is added to the grid below.

    • From the Site Status and Site Status Attribute fields, select the desired values.
    • From the Status Start Date and Status End Date fields, select the desired start date and (if there is one) end date for the status .
    • In the Notes fields, enter any notes for the status change (optional).
  4. Click Save to save the change and populate the entry's ID values.

To Modify a Status Entry for a Site

  1. Open a Site Manager Document that contains the site you wish to modify.
  2. Select the site in the document grid. The Details pane displays.
  3. Click the Site Status tab, and modify the entry as desired.
  4. In the Entry toolbar, click Save.

To Delete a Status Entry from a Site

  1. Open a Site Manager Document that contains the site you wish to modify.
  2. Select the site in the document grid. The Details pane displays.
  3. Click the Site Status tab, and select the status you want to delete.
  4. In the Entry toolbar, click Delete to remove the entry from the grid.
  5. In the Entry toolbar, click Save.
