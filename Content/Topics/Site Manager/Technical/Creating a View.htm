Creating a View

In order to reference metadata, it must either be contained in a specific view, or referenced within an existing table. An initialization procedure (INIT) should also be created to populate this newly created view. A primary key must be included within the view in order for functions in the tbl_Common_Filter_MetaViewData table to work.

Example

Example for uvw_Site_SiteCapacity:

SETANSI_NULLSON

GO

 

SETQUOTED_IDENTIFIERON

GO

 

 

CREATEVIEW [dbo].[uvw_Site_SiteCapacity]

AS

SELECT

si.SiteID

,SUM(c.SellingCapacity)AS TotalSellingCapacity

,SUM(c.StockroomCapacity)AS TotalStockroomCapacity

FROM tbl_Site si

LEFTOUTERJOIN tbl_Site_Capacity c

ON c.SiteID = si.SiteID

GROUPBY si.SiteID;

 

GO

After you create and populate the view, it can be referenced in the tbl_FilterType table.

Initialization

Initialization procedures must be created and/or run to populate the tbl_Common_Filter_MetaViewData table.
