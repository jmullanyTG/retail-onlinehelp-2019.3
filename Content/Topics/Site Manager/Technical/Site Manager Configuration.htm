Site Manager Configuration

The Configuration button is located at the top on the right side of the Site Manager Document screen in the Site Manager solution. Use this button to set up a variety of views.

When configuring widgets for the site manager, be sure to include the SiteID parameter in the configuration.

To Set Up a View

  1. From the Management tab, click Site Manager. The Item window displays.
  2. In the Items window, click Cancel to close the window.

  3. Click the Configuration button at the top of the screen. The View Maintenance window displays.

  1. From the View Maintenance window, select the views included as blades in the site manager detail section, and click OK.
If you add or remove a view, you must refresh the plugins or exit Demand Management and relaunch to see the changes. Each site detail blade is comprised of a view that can be set with individual user permissions to determine access.
