Example: Setting Global Priority

In this example, you will set the global priority values for the new stores Calgary and Nashville.

To Set a Global Priority Value to a New Store

  1. From the Management tab, click Site Manager. The Items window displays.

  2. From the Filter field, select the filter that contains the sites you want to manage, and click OK. he Site Manager Document displays the two new sites Calgary and Nashville.

    In this example, the Identify New Stores filter is used.

  3. Click on either store in the grid. The Details pane displays.
  4. In the Details pane, click the Global Site Priority tab.

  5. In the Global and Group Priority grid, locate the Nashville store in the and change its Global Priority value to 43.

  6. In the Global Site Priority toolbar, click Save. Nashville is now ranked 43rd in the Global Site Priority list.
