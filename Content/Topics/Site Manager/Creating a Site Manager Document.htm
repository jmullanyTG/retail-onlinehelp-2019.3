Creating a Site Manager Document

From Site Manager, you can create a Site Manager Document to manage your sites.

To Create a Site Manager Document

  1. From the Management tab, click Site Manager. The Items window displays.

  2. From the Filter drop-down menu, select the filter that contains the sites you want to manage, and click OK. The Site Manager Document displays.

  3. In the local Columns toolbar, click Choose to select additional fields to display, and click OK. The Site Manager Document displays.

  4. Review the information and compare details across, or select individual sites in the grid to view additional details in the Detail pane.

    :

    If you change any settings in the Details pane, click Save to save the changes.
