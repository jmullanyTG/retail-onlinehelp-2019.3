Site Capacity

The hierarchy used to populate the product dimension in the Site Capacity widget is determined by the Site.Capacity:Product Hierarchy system policy.

From the site capacity view, you can view and update site capacity information in Demand Management. This data is used for reporting and filtering only.

Field Description
Hierarchy Level

The specified hierarchy level at which capacity is measured for the site. The hierarchy used is controlled through a system policy setting.

Level

The level value in the product dimension. This value associates directly with the hierarchy levels.

Selling Capacity Showroom capacity in quantity of units.
Stockroom Capacity

Stockroom capacity in quantity of units.

Minimum Capacity

Minimum capacity for the total site.

Maximum Capacity

Maximum capacity for the total site.

Active

A flag that identifies the active status of the capacity constraint.

Created Date

Date of creation for the capacity entry.

Created By User credited with the creation.
Last Updated

Date of the most recent update to the capacity entry.

Last Update By

User credited with the most recent update.

You can set selling capacity, stockroom capacity, minimum capacity, and maximum capacity in the Site Capacity view, as well as capacity settings active status. This information does not affect order creation, but may be used to drive filtering.

Site-Specific and Aggregate Capacity by Product Level

You can define site capacities at a desired level in your product hierarchy (total store, department, style, etc.). Demand Management displays the aggregated capacity at the higher levels.

To Set up Site Capacity at the Hierarchy Level

  1. From the Management tab, click Site Manager. The Items window displays.
  2. From the Filter drop-down menu, select the Site.Capacity:Product Hierarchy filter that you created in a previous example, then click OK. The Site Manager Document displays.
  3. In the local Columns toolbar, click Choose and select Site Capacity.
  4. Click OK. The Site Manager Document is update with the new columns.
  5. From the Site Capacity tab, click Parameters. The parameters for site capacity display.
  6. From the Hierarchy Level drop-down menu, enable the hierarchy levels you want to use, with the lowest level enabled being the level at which site capacity will be entered.
  7. When finished, click Save.

To set up site capacity at the system policy level, repeat the procedure for setting up site capacity at the hierarchy level. From the Hierarchy Filtering view, enter the ID of the hierarchy form.

After the setup is completed, you can enter site capacities for each of your locations on the Site Manager’s Site Capacity Detail tab.

After site capacities are entered, you can view aggregate capacities by selecting higher hierarchy levels on the tab Site Capacity tab.
