Example: Validating Site Data

In this example, you will create a Site Manager Document that displays all stores added in the past 60 days. You will use this document to verify that the information received from the ERP system for each site is valid.

To Validate Site Data

  1. From the Management tab, click Site Manager. The Items window displays.

    If the Items dialog box does not automatically open, click + Site Manager Document on the toolbar to open it.

  2. From the Filter drop-down menu, select the Identify New Stores filter, and click OK. The Site Manager Document displays the newly added stores Calgary and Nashville.

  3. In the local Columns toolbar, click Choose and select Site Type, Site Status, and Site Group and click OK. The Site Manager Document displays.

  4. Review the site groups, attributes, status, and other data fields for each new site to ensure they are correct. Be sure to also click the tabs in the Detail pane to ensure the values they show are as expected.
