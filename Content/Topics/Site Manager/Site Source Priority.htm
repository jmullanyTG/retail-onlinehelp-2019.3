Site Source Priority

The site source priority view site specific sourcing priority for other sites which are classified as stores. This is a read-only view that is populated by the Supply Chain module. There is a system policy used to determine the supply chain that populates this view. All active store sites which are not included in the site’s priority will be struck-through.

  • Source Site Code: the site code for the store.
  • Source Site Type: the site type (store).
  • Source Site: the name of the site.
  • Source Site Priority: the priority ranking as established by the supply chain.
  • Default Site Lead Time Days: the default lead time (set at site level; product agnostic).
