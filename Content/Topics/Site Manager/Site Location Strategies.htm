Site Location Strategies

In Allocations, store type sites are grouped into location strategies to determine allocation quantities; the membership of a specific site is not exclusive to a single location strategy. From the Site Location Strategies tab, you can view and maintain the membership of a site in active location strategies.

Field Description
Is Included? Indicates if a site is included in a location strategy. You can modify the membership of a site in. Any updates to location strategies require calculation of all linked allocation strategies. Distribution Centers cannot be included in location strategies.
Strategy Location Name The name of the location strategy.
Strategy Location Description The description of the location strategy.
