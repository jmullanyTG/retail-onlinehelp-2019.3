DC Source Priority

The DC Source Priority tab displays the site specific sourcing priority for distribution centers. This is a read-only view that is populated by the Supply Chain solution. There is a system policy used to determine the supply chain that populates this view. All active DC sites that are not included in the site’s priority display struck-through.

Field Description
Source Site Code Site code for the distribution center.
Source Site Type The site type (DC, DC/Store).
Source Type The name of the distribution center site.
Source Site Priority
  • The priority ranking as established by the supply chain.
    • Default Site Lead Time Days The default lead time (set at site level; product agnostic).
