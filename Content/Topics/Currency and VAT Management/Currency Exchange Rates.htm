Currency Exchange Rates

The Currency Exchange Rates view allows users to maintain both global exchange rates and time-based exchange rates from and to all currencies defined within Demand Management. At a minimum, exchange rates should be maintained globally for a single “From” currency, being the home currency used in your assortment plans and as many “To” currencies as you have assigned to the sites within the cluster sets in those plans.

This screen can be accessed by navigating to System Utilities > Views > Currency > Currency Exchange Rates.

Only doing business in one currency

If your company only does business in a single currency, then set a single conversion entry “From” and “To” your home currency with a Conversion Factor of 1 to indicate no change.

Fields

Field Description
Currency From Sets the “From” currencies to be worked with in the view
Currency To Sets the “To” currencies to be worked with in the view

Toolbar buttons

Button Description
Parameters Shows or hides the Parameters area of the view
Execute updates the view with a conversion factor grid for the Currency From and Currency To selections

Default Currency Exchange Rates Tab

This tab allows the user to manipulate the field layout by dragging and dropping fields in the Filter, Row, Column, and Data areas of the grid.

Functionality

  • Export: Exports and saves the Pivot Grid in different formats including Microsoft Excel and Adobe PDF
  • Open: Opens the Pivot Grid in different formats including Microsoft Excel and Adobe PDF – does not require a save prior to viewing in the desired format
  • Graph: Displays a graph and any data series selected by the user

  • Fields: launches the Fields Customization dialog with two tabs. Each tab contains a search bar in which you can enter text to more quickly locate the fields you wish to manipulate. The fields shown in the view will be limited to only those containing the text you have entered.

    • Simple tab: fields can be dragged and dropped between Filter, Row, Column, and Data areas of the Pivot Grid

      Dragging fields to the None area hides them in the layout.

    • Advanced tab: Manipulate the layout by selecting the desired location of each field from the Pivot Area drop downs next to each. An Index can be specified within each area to define the order in which the fields display.

      Changes made to the Assortment Plan Defaults By Time layout can be saved for future use. After making changes and clicking OK in the Fields Customization dialog, right click anywhere in the Pivot Grid and choose Save Layout.

Restoring layouts

Restore a custom layout by right-clicking anywhere in the Pivot Grid, choosing Restore Layout, and selecting the layout to restore.

  • Save: save all changes made to conversion factors in the current view since the last save
  • Cancel: reverts all changes to conversion factors in the current view since the last apply to their previously saved values
  • From Currency Description: text description of the From Currency
  • From Currency Code: unique system code for the From Currency
  • To Currency Description: text description of the To Currency
  • To Currency Code: unique system code for the To Currency
  • Conversion Factor: user editable decimal (1.0001); sets the exchange rate from the From Currency to the To Currency

    A Conversion Factor of .88 from USD (US Dollars) to GBP (UK Pounds) means a sale of $1,000 USD buys 888 GBP.

    Setting a Conversion Factor of 1 indicates no change.

Temporal Currency Exchange Rates Tab

This tab allows users to enter exchange rate changes with specific Start and End Dates which is useful for planning for future economic fluctuations ahead of time without having to alter the default exchange rates. Temporal Currency Exchange Rates supersede rates entered on the Default tab for the time frames in which they are active.

  • New: adds a new line for entering a temporal exchange rate
  • Delete: deletes the selected temporal exchange rate lines

    users are not asked to confirm deletion of temporal exchange rates, so be certain you want to remove them before clicking Delete

  • Fields: opens the Customization dialog in which users can drag fields to or from the tab’s header area and this dialog to customize the view

    • Custom layouts can be saved/restored by right-clicking in the Pivot Grid and selecting Save Layout or Restore Layout, respectively

  • Save: saves all new temporal exchange rate lines and changes made to existing lines since the last save
  • Cancel: removes all new temporal exchange rate lines and reverts all changes since the last save back to their previously saved values
  • Currency From: the currency to convert from for the line
  • Currency To: the currency to convert to for the line
  • Start Date: the start date for the temporal currency conversion

    creating temporal currency conversions in the past WILL NOT have any effect on actualized periods

  • End Date: the end date for the temporal currency conversion; not required

    not setting an End Date for a conversion means it will persist into the future until one is set or another conversion is entered in the future – this effectively makes the End Date for the conversion the day prior to the Start Date of the new conversion

  • Conversion Factor: user editable decimal (1.0001); sets the exchange rate from the From Currency to the To Currency

    I have a Conversion Factor of .88 from USD (US Dollars) to GBP (UK Pounds), thus a sales of $1,000 USD will equal 888£ GBP
