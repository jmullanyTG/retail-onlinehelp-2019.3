Series

Seeing the graphical representation of the different series related to the forecast gives users insight into the meaning of the data.

  • Manual Forecast – Will be dark red if a forecast has been manually changed.
  • Effective Forecast – Will be the values passed along to the replenishment engine via the Inventory Manager. Should align with the system forecast (blue series) before any changes are made.
  • System Forecast – The system generated forecast produced by the Demand Management forecasting engine. The values past the current date will represent the values produced by the winning forecast model. Any series values preceding the current date may be viewed as the “model fit”, a combination of the simulation and backtrack.
    • Backtrack – The backtrack series is considered to be “optimistic” in that it will correct itself to be closer to the historical series for each history period.
    • Simulation – The simulation is considered to be “pessimistic” in that it will not correct itself in any way over the historical periods.
  • Original Forecast – The yellow bars (not the yellow dotted-line) will represent the original system generated forecast. As changes are made, they will remain for comparison.
