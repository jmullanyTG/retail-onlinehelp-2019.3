Forecast screen toolbar

The toolbar contains the following buttons.

Button Description

Save

If any changes are made to the forecast (re-forecast, manual forecast adjustments, etc), the save button will become available. If the forecast alterations are intended to be kept, users should use the save button to enable these changes.

Undo

If any changes are made to the forecast (re-forecast, manual forecast adjustments, etc), the undo button will become available. If users wish to undo the changes, they may do so using this button.

Change All

Allows users to change multiple lines (either rows or columns) within the grid. They may be set to a specified value, increased or decreased, copied from another column or row, or split across sever other columns based on a percent contribution.

Forecast

Generates a forecast for a selected item. Selecting Forecast launches a window similar to the following when forecasting an item for the first time.

Lock/Unlock
  • Lock. Allows users to include or exclude certain forecasting methods from being run for the inventory item. A “lock expiration” must be set to determine the period of time for which the lock is intended to cover.
  • Unlock. If a forecast has had a method lock applied, the “unlock” button will appear, allowing users to remove the method lock.

Reviewed

As part of a business flow process, the reviewed button may be used to show that particular forecast for an item has been approved by the appropriate user. Clicking “review” will also populate the current date in the “reviewed” field.

Notes

Selecting the Notes button launches the Notes Viewer, which will allow users to communicate any information related to the particular inventory item as it relates to the forecast.

Causals

Launches the maintain item causal links window, which will allow users to include or exclude any causals which have been created in the Demand Management Causals module.

Inventory Manager

Launches an Inventory Manager view for the inventory item to bring additional pertinent information to the user’s attention.

Forecast All

The “forecast all” button will allow users to run a re-forecast for all items within the filter. For very large filters, this may take some time.

Settings

Launches the “visible bucket period limits” window. This will determine the number of periods which will be displayed in both the graph and grid sections of the forecast viewer. Depending on the period (bucket) type, be it daily, weekly, or monthly, the visible bucket period limits may be adjusted to take a bigger picture look at the inventory item, or narrow the focus if so desired.

Replace

If the users have determined a need to replace the method by choosing a different method from the drop-down list, the “replace” button will appear. Selecting this will replace the effective forecast
