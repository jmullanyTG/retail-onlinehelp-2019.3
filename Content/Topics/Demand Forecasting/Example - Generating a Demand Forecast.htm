Example - Demand Forecast

In this example, you will generate a demand forecast for the item High Collar Angelina Jacket.

To Generate Demand Forecast for an Item

  1. From the Management tab, click Forecasts. The Items - Level View window displays.

  2. From the Filter drop-down menu, select the New Items (Creation <=14 Days) filter, and click OK.

  3. On the Forecasts screen, select the new item, High Collar Angelina Jacket_Grey - L:

    1. Click the Items list.
    2. In the Item Master Code filter, enter 708.

    3. In the Channel column, double-click the row that contains Retail.

  4. Create a demand forecast for the new item:

    1. In the Forecast toolbar, click Forecast.
    2. In the Forecast Task window, click OK to star the forecast tournament.

    3. When the forecast is complete, it displays on the Forecast screen.
