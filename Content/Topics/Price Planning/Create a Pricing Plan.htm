Create a Pricing Plan

You create Pricing Plans on the Home screen of the Price Manager module. Additional products and sites may be configured at a later time.

The level at which Price Actions will be created for a new Pricing Plan is dictated by the system policy Product Pricing Level and is indicated on the Pricing Plan Information tab as follows:

The possible values are:

  • Variant by Site
  • Item by Site

The Create New Plan dialog

Within the Price Manager Home screen, click New to create a new pricing plan, enter the required fields and click OK.

Field Descriptions

Field

Description

Plan Name

The display name of the pricing plan.

Plan Description

An optional description of the pricing plan.

Start Date

Select a future date that associated pricing actions will be defaulted to. The system will default the start date and restrict selection of an earlier date, by a ‘globally’ configured number of days forward from today. Special permissions (i.e. Emergency Pricing) will allow you to override this restriction and allow selection of tomorrow at the earliest.

When modifying the Plan Start Date, the system will restrict moving the date forward any further than by any Pricing Action Start Date.

:
Original Plan Start Date = 01/01/2016
Earliest Price Action Start Date (already exported) = 02/01/2016
Result: System will not allow a new Plan Start Date greater than 02/01/2016

Variant Filter

Select a preconfigured filter form the drop down or alternatively click Maintain Filter to configure a new one to designate the products to be used within the pricing plan's action(s). This will be the initial set of products to be added to the price action. Additional products may be added or removed at a later time.

Site Cluster Set

Site Cluster Sets are available based on Site Clustering configuration.
Product Pricing Level

A drop down menu with the following options:

  • Variant Site Plan
  • Item Site Plan

This field only displays when system policy Product Pricing Level is set to Variant\Site and Item\Site Pricing.
