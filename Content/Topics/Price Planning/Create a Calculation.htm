Create a Calculation

Within the Price Manager Configuration Calculation screen, click New to create a new Calculation or Calculation Rule, enter the required fields and click Save.

Field Description

Calculation Name (Required)

Enter a descriptive label to summarize the purpose of the Calculation or Rule.

Calculation Description (Optional)

Enter a summary of the Calculation or Rule.

Calculation Method (Optional)

Select from a list of preconfigured database stored procedure for more complex mathematical formulas or validations. Selection in the ‘method’ field will override any entry in the ‘formula’ field.

Calculation Formula (Required/Optional)

Enter the mathematical expression for the system to evaluate. Configured variables can be found in the right panel. Selection in the ‘method’ field will override any entry in the ‘formula’ field.

Calculation Example:

Retail Sales Price is % Above Cost1
Variable1 = (1+Variable2)* Variable3
Variable1= [RSP]
Variable2 = [PercentAbove]
Variable3 = [Cost1]
[RSP] = (1+[PercentAbove])*[Cost1]
$5.988 = (1+.20)*$4.99

The system will populate [RSP] to the ‘targeted output field’ and the system will continue validation and calculation of the Price Action Strategy.

Targeted Output Field (Required)

Select from the list of configured Variables (i.e. user input only) that the configured mathematical formula will populate. Output fields cannot be specified for ‘validation’ types.

Rounding Rule (Optional)

Select from the list of configured Rounding Rules for the system to apply to the Calculation. Rounding Rules cannot be applied to ‘validation’ types.

Calculation Active (Optional)

Unmarking the check box, will remove the ability to configure the Calculation in any new Price Strategies.
