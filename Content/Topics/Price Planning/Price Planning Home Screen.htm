Price Manager Screen

You can view pricing plans and their related price actions on the Price Manager screen.

You can perform the following actions from the Home tab.

Action

Description
Open a pricing plan Double-click on a plan in the Plans grid to open the plan in a new tab. (You can also select the plan and click Edit on the Plan toolbar.)
Create a new pricing plan Click New on the Plan toolbar to create a new pricing plan.
Delete a pricing plan Click Delete on the Plan toolbar to delete a selected pricing plan.
Open a price action Double-click on a price action in the Price Actions grid to open the action in new tab. The action will be opened in the context of its parent plan.

Validation failures

The Price Actions grid has a column labeled Validation column. This column is populated by the system with Validation Severity icons that let you quickly distinguish between Validation failures that have been reviewed and Validation failures that have not yet been reviewed or resolved by modifying the Price Action to comply with corporate pricing guardrails.

Validation Severity icons

The following are the Validation Severity icons for failures not yet reviewed:

Validation Severity icon Description
Information Provided
Warning Encountered
Error Found

The following are the Validation Severity icons for failures that have been reviewed:

Validation Severity Icon Description
Information Reviewed
Warning Reviewed
Error Reviewed

For more information, see Price Action Validation Icons, the Validation Log, and Overriding Validation Errors.
