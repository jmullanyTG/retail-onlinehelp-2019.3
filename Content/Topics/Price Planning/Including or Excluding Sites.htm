Re-including or excluding Sites and Site Clusters in a Pricing Plan

To re-include or exclude Sites or Site Clusters in a Pricing Plan, navigate to the Sites pane on the Pricing Plan Information tab.

The Include and Exclude buttons are disabled by default if:

  • any Price Actions exist as the Pricing Plan is the main template of Sites and Site Clusters for Price Actions that needs to be retained.
  • no Sites or Site Clusters are selected.

The Include button is enabled if an excluded Site or Site Cluster (indicated by a strikethrough) is selected. The Exclude button is enabled if an included Site or Site Cluster is selected.

Once you click either button, the system evaluates the Pricing Plan Start Date (displayed in the Start Date field) and determines if a new Site Cluster Set Snapshot exists. If a new Site Cluster Set Snapshot does not exist, then any inclusions or exclusions of Sites and Site Clusters are saved to the Pricing Plan.

If a new Site Cluster Set Snapshot does exist, then a message dialog displays to inform you that an updated Site Cluster Set exists and will be loaded. Once you click OK, the Site Cluster grid is refreshed with the Site Cluster Set Snapshot applicable to the Pricing Plan’s Start Date and exclusions and re-inclusions must be re-selected.

For more information on Snapshots, see Creating and Managing Financial Plans.

For more information on Snapshots, refer to the online help.

Excluding Sites and Site Clusters

In the Sites grid:

  1. Select the check box by the name of the Site Cluster to be excluded or select check boxes within a Site Cluster to exclude particular Sites.

    The Exclude button is enabled.

  2. Click Exclude.

    The screen automatically refreshes and the names of excluded Site Clusters and Sites are denoted by a strikethrough.

    The system enforces that there be at least one Site Cluster and Site included in a Pricing Plan. Therefore, the following message displays if you attempt to exclude all Site Clusters.

    Clicking OK closes the dialog and refreshes the screen allowing you to continue your modifications.

Re-including Sites and Site Clusters

In the Sites grid:

  1. Select the check box by the name of the excluded Site Clusters (indicated by a strikethrough) to be re-included or select check boxes within a Site Cluster to re-include particular Sites.

    The Include button is enabled.

  2. Click Include.

    The screen automatically refreshes and the names of the re-included Site Clusters and Sites are no longer denoted by a strikethrough.
