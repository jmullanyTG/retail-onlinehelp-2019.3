Modify Price Action Statuses

The Price Action Status provides the ability to drive corporate configured processes and associated permissions by roles and users, for each Status, as well as provide communication between Statuses that can be reviewed to understand the lifecycle of the Pricing Action.

Changing Price Action statuses

You can change Price Action statuses by clicking the Change Status(es) button to display a drop-down menu that allows you to change the status of a single Price Action or change the status of multiple Price Action at one time. The ability to change multiple Price Action Statuses at the same time so that the Price Action may proceed in their life-cycle, whether for approval or to send to downstream systems, greatly reduces the repetition associated with opening and changing the Status for each and every Price Action.

The menu options are:

  • Change Status
  • Change Multiple Statuses

Change Status

The Change Status menu option is based on the Price Actions selected in the List grid. Selecting this option opens the Change Status dialog.

This menu option is enabled when:

  • The Price Action Start Date is today or in the future.
  • The Price Action State is not Calculation Required.
  • The user has User Rights to modify the Price Action status in its current Workflow Status State.
  • The User has permissions to the next Workflow Transition.

Change Multiple Statuses

The Change Multiple Statuses menu option is based on ALL the Price Actions displayed in the List grid. Selecting this option opens the Change Statuses dialog.

Field Description
Price Action ID (Read Only) The Price Action unique identifier
Price Action (Read Only) The Price Action name
Workflow Name (Read Only) The Price Action Workflow
Price Action Status Selecting this field displays a drop-down with the next available Status for the assigned Price Action's Workflow that the user has been provided access.
Owner

Selecting this field displays a drop-down populated with all Users and Roles that are not disabled and not deleted. The list is filtered by Users that have the right to edit Price Actions.

The purpose of changing the Price Action Owner is to assign the Price Action to another user for additional work or review and approval.
Message An editable text field used for capturing the reason for the Status change or a message to the newly selected owner.

This menu option is enabled unless any one of the following criteria is true:

  • All Price Actions have a Start Date that is in the past.
  • All Price Actions have a State of Calculation Required.
  • The User does not have permissions to modify Price Action status during the current Status State for all Price Actions.
  • The User does not have permissions to the next Workflow Transition, for all Price Action.
