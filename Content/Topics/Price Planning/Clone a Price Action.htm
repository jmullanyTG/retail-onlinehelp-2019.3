Clone a Price Action

You can clone a Price Action by navigating to the Price Actions screen. Within the Price Manager screen, double-click a row in the Price Actions grid. Click the Actions tab, next to Information, which will load the Price Actions screen.

Cloning a Price Action

  1. Click Clone.

  2. Complete the fields as required.

    The system appends (Copy) to the original Price Action Name. Overwrite with a new name as required.

  3. Click Ok to create a new Pricing Plan.

    The Price Action grid refreshes with the newly cloned Price Action selected.

Clicking Cancel discards the creation of the new Price Action and closes the Clone Price Action dialog.

Field Descriptions

All fields are editable.

Field Description
Name The original Price Action name.
Description The original Price Action description.
Start Date

The original Price Action Start Date. If the start date is in the past, then:

  • if the User has the Right Emergency Price Action the system will default the Start Date to today.
  • if the User does not have the Right Emergency Price Action the system will default the Start Date to tomorrow.
Strategy

The original Price Action Strategy.

 

  • If not modified the system will copy over the Strategy Variable values and any lower level exception overrides.

  • If modified, the system will NOT copy over the Strategy Variable values nor any lower level exception overrides.
Workflow The original Price Action Workflow. The System will apply the first Status for the selected Workflow.
Variant The system applies the selected Variants from the original Price Action. Additionally, the system applies any product exclusions across all levels.
Site Clusters The system will display the selected Site Cluster(s) from the original Price Action. If a Cluster is added, that was previously not included, then the system will apply the default Strategy values for the appropriate currency.
