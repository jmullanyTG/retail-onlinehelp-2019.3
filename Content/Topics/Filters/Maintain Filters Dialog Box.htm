Maintain Filters Dialog Box

The Maintain Filters dialog box is the starting point for creating and maintaining filters.

You can open the Maintain Filters dialog box by clicking:

  • Filters on the Maintain tab.
  • The Maintain () button next to a filter selection list.

From the Maintain Filters dialog box, you can perform any of the following actions.

Action Description
Create a New Filter Creates a new filter from scratch.
  • Choose Clone From Existing Filter to create a copy of an existing filter. You must have permission to view a filter to select it from the Filter list.
Update The Following Filter Opens the filter selected in the Filter list for editing.
Delete The Following Filter(s) Deletes the filters you select.
Export/Import Filters These options are used to move data from one instance of Demand Management to another and are typically reserved for technical Demand Management consultants only.
