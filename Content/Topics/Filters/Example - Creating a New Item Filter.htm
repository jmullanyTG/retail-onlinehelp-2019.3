Example - Creating a "New Items" Filter

In this example, we will create a filter that identifies all items created within the past 14 days. Of course, you are free to change the number of days to any value that best suits your purpose.

Instructions

  1. In the Management tab, click Filters.
  2. In the Maintain Filters dialog box, select Create a New Filter and click OK.
  3. In the Select Filter Entity type list, select Inventory and click OK.
  4. In the Maintain Filter dialog box, enter the following information:
    • Filter Name: New Items (Creation < 14 days).
    • Filter Notes; Add a note.
    • Filter Group: select a filter group.
    • Share this File: select a sharing option.
  5. In the Criteria section, click New.
  6. In the Setup Filter Criteria form, select Criteria where all of the following conditions must be met.

  7. In the Condition section, click New.

  8. In the Filter Condition dialog box, enter the following:

    • Field Select: Days From Creation in the Inventory Planning group.
    • Condition: Select Smaller or Equal To
    • Days From Creation Smaller or Equal To: Enter 14

  9. The new filter is ready to use.
