Managing User Rights and Roles

The Access Notification System Maintenance global permission allows a Super User to access and manage user permissions from the Notification Subscription screen.

To Create a New User

  1. From the Maintenance tab, click Users. The Navigation Menu displays.
  2. From the Navigation Menu, click Create. The User Information window displays.
  3. Enter the following information on the respective tab:
  • User Details: user name, description, password, etc.
  • Contact Details: telephone number, email address, fax number, etc.
  • Roles: click the > and < buttons to add or remove roles for the user.
  • Rights: click the > and < buttons to add or remove rights for the user.
  1. Click OK. The new user is created.

To Change User Information

  1. From the Users tab, click a user name on the right panel next to the Navigation Menu.
  2. From the Navigation Menu, click Edit. The User Information window displays.
  3. Make the necessary changes.
  4. Click OK. The user information is updated.

To Delete a User

  1. From the Users tab, click a user name on the right panel next to the Navigation Menu.
  2. From the Navigation Menu, click Delete. The Delete User window displays.
  3. Click OK. The user information is deleted.

To Clone a User

  1. From the Users tab, click a user name on the right panel next to the Navigation Menu.
  2. From the Navigation Menu, click Clone User. The User Information window displays.
  3. Review the user information.
  4. Click OK. The user profile is cloned.

To Assign or Change a Role to an Existing User

  1. From the Users tab, click a user name on the right panel next to the Navigation Menu
  2. From the Navigation Menu, click Assign Roles. The User Information screen displays.
  3. From the Roles tab, click the > and < buttons to add or remove roles for a user.
  4. Click OK. The user role is updated.

To Assign or Change Rights to an Existing User

  1. From the Users tab, click a user name on the right panel next to the Navigation Menu.
  2. From the Navigation Menu, click Assign Rights. The User Information window displays.
  3. From the Rights tab, click the > and < buttons to add or remove rights for a user.
  4. Click OK. The user rights are updated.
