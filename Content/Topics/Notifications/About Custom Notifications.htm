About Custom Notifications

You can set up e and receive custom notifications through the Default delivery method.

Custom Notifications are initiated by a scheduled task in the Demand Management Job Manager or through a customized stored procedure that can only be configured by a Demand Management Professional Service or Support.

To Create a Custom Notification

  1. From then Maintenance solution, click Notification Maintenance. The Notification Maintenance screen displays.
  2. Under the respective solution, click the [+ Add Custom Notification] row. A new row displays.
  3. In the Notification Properties panel, do the following:
    1. Enter a notification title.
    2. Select the notification type.
    3. Enter the notification message.
  1. Click Save. The custom notification message is created.
To initiate the custom notification, see Initiate Custom Notification Configuration.

Field

Description

Solution (Required)

The notification solution depends on where the custom notification is initiated.

The solution selection:

  • Determines which icon will be used in the notification message that is sent to users.
  • Can be used to sort messages in your Notification Inbox.

Title (Required)

The notification title describes the purpose of the message that is sent to users. Follow the corporate standards naming conventions used in your company, if applicable. The notification title is not displayed in the message sent, but it is updated in the Notification subscription window.

Type (Required)

The notification type displays an icon according to the message type selected. The system defaults all custom notifications as a General Notification type.

The notification types available are as follows:

  • Assignment
  • Completed
  • Error
  • General
  • Information
  • New Task
  • Warning

Message (Required)

The notification message is the content in your message, which also displays in your Notification Inbox.

UUID (Read Only)

The Universal Unique ID number is a system-generated identifier used when you configure the notification to be initiated in the Demand Management Job Manager or through Customized Stored Procedure (CUSP hook).
