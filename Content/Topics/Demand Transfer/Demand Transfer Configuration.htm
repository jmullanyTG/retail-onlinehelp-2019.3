Demand Transfer Configuration

To include demand transfers in an item’s replenishment strategy, perform the following configuration tasks:

Module Task
Inventory Manager Select the Consider Demand Transference option for the item.
Job Manager Ensure the (DF) Calculate Fulfillment Split Factor task is added to daily/weekly jobs. This task must run after DF tasks but before RP tasks.
Inventory Policy If the item is sold at a showroom, configure the showroom Demand Transference options for the item's related inventory policy.

Once configured, demand transfers will be included in order recommendations run by the Demand Management replenishment engine.

After orders have been generated, you can see the split factors for an item-site combination from the Forecast Fulfillment Split Factors tab in the Inventory Manager's Detail pane.
