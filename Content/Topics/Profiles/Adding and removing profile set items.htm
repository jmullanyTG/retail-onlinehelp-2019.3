Adding and removing profile set items

Adding and removing profile set items is done once the profile set is open in a tab using the Add and Remove buttons.

Clicking the Add button launches the Add Profile Set Items window. The Remove button removes the highlighted item from the set.

Selecting Choose Curve displays the Curve Manager window. Select the curve to be used and click Ok.

Once the curve is selected, users should enter the date ranges for the profile start, profile sample, profile effective, and value for the profile evaluation period

 

Once completed, select Next.

Next, enter the Group Types for the profile item. Depending on how groups have been configured, this will vary from one implementation to another.

Set the included or excluded groups for each group type.

Once completed, click Ok, then Next on the group types window.

Once the group types have been set, add the individual profile set items.

Click Finish to populate the profile set items.

Each item line can be modified for dates and volumes. The effective dates are the time period during which the profile curve will be applied to the projected demand.
