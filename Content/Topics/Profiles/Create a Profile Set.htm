Create a Profile Set

A profile set is a collection of profiles and profiled items. Building a profile set is an important first step in creating and applying profiles.

Fields

  • Name – The name for the profile set.
  • Description – The optional description for the profile set.
  • Group – The parent folder used to assist with organization of profile sets.
  • Filter – The filter which will be applied to determine the items within the profile set (items to be profiled. This must be a static filter as any changes made to the filter will cause the Profiles Module to exclude it from the sets (this is intended to maintain the integrity of the profile).
  • Security – The security settings will affect which users may or may not edit the profile.
