Profile Set Items grid field reference

The profile set items details inventory specific information about how the profile is being applied. This provides an overview of the inventories which are affected by the profile and in which way.

  • Product – The item master code for the inventory.
  • Product Description – The description tied to the item master code.
  • Location – The site code for the inventory.
  • Location Description – The description tied to the site code.
  • Curve Changed – If the selected curve has changed, this check box will become flagged. Not a user editable field.
  • Re-Apply Curve – If the curve needs to be re-applied, users should check this box.
  • Curve Name – The selected curve (from the Curves Module/curve management screen). May be changed by using the “Choose Curve” button.
  • Causal Shift – Controls whether the total sales value of the model is held or increased based on the application of the causal.

    • Reshape - Holds total sales and will decrease the non-causaled period contributions (assuming a positive causal effect). It "reshapes" the whole profile to accommodate the causal factor.

    • Lift - Does not hold total sales and the impact of the causal is only recognized on the causaled period. It "lifts" the affected periods.

  • Profile Start – The date from which the profile evaluation period is to start.
  • Effective Start – The actual start date for the application of the profile (considers any and all constraints).
  • Effective End – The actual end date for the application of the profile considers any and all constraints).
  • PEP (Profile Evaluation Period) – The number of periods to use to evaluate the effectiveness of the profile curve (counts forward from the start date).
  • Sample Start Date – Start date of the sample period.
  • Sample End Date – End date of the sample period.
  • Sample Volume – Total sales volume during the sample period. The sample data will be used to determine the magnitude of the profile.
  • Sample Average – Average sales (per period/bucket) during the sample period.
  • PEP Data Points – Number of data points during the performance evaluation period. Includes data points from the current period historically back to the specified number of specified (PEP value), but not further than the effective start date.
  • PEP Sales Volume – Total sales volume during the performance evaluation period.
  • PEP Profile Volume – Total sales using the profile curve.
  • PEP Sales Average – Average sales (per period/bucket) during the performance evaluation period.
  • PEP Profile Average – Average sales of the profile (per period/bucket) during the performance evaluation period.
  • Profile Override (%) – Override percentage for the profile ((PEP Sales Volume – PEP Profile Volume) / PEP Profile Volume * 100)
  • Apply Override (%) – Will be checked if it is intended to apply the override percentage.
  • Override Effective Volume – Sales volume for the effective profile period if the override is applied.
  • Override Effective Average – Average sales (per period/bucket) during the effective profile period if the override is applied.
  • Effective Volume – Sales volume over the effective period.
  • Effective Average – Average sales (per period/bucket) over the effective period.
  • Profile Volume – Sales volume for the entire profile period.
  • Profile Average – Average sales (per period/bucket) for the entire profile period.
