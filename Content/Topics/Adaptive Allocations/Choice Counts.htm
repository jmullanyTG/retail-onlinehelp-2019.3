Choice Counts tab

The choice count results tab displays the calculated choice count by store, alongside the historical choice count for the same store as used in the sampled time frame. This can be used for reference when validating choice count results. All stores within the location strategy display.

Given similar financial targets, the choice count model will likely yield fewer counts than the comparative historical count due in part to the qualification thresholds.

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Choice Count – The calculated choice count result.
  • Historical Choice Count – The historical choice count used within the calculation.

Viewing Choice Counts in a graph

You can view the results as a graph by toggling the view using the Graph button within the Scenario Details menu section.
