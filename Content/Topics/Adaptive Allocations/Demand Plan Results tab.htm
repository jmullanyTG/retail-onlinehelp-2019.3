Demand Plan Results tab

Once a demand plan has been tied to a scenario plan (and transitively, a location strategy) the by store demand plan values may be calculated and displayed.

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Need Target – The calculated need target for the store. Will display in “base” currency and is representative of total cost of goods sold (COGS) or retail sales target based on the specified accounting method.
  • Site Groups 1-20 – Additional site grouping information (i.e. channel) may be returned to provide reporting options.
