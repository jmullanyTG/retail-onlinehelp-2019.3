Creating Rules using the Allocation Rule Wizard

The Allocation Rule wizard guides users through the steps necessary to create or edit allocation rules. In the first screen, users will name the rule, choose the type and select the appropriate rule groups.

Users will be prompted to include appropriate inventory, itemmaster (product hierarchy), or site groups.

Once the possible groups have been selected, the included group values must be determined.
