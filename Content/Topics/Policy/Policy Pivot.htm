Policy Pivot

Policy Pivot is used to set up a policy by filter or hierarchy level. You can set up active policy settings through mass changes in the Pivot grid.

Access to the Policy Pivot screen depends on user access rights. Because small changes can affect critical values across large segments of the organization, it is highly recommended that only users with a full understanding of the consequences be granted access to the feature.

The parameters used to display information in the Pivot grid are controlled by user-assigned filters. If items or locations are excluded from a user’s assigned filters, but kept in the hierarchal segmentation or selected filter, the inventories will not appear in the pivot grid and the inventory policy settings will not be changed.

To Access the Policy Pivot Screen

  1. From the Execute tab, click Policy. The Policy Task screen displays.
  2. Click the Pivot icon on the right of the screen. The filter parameters display.
  3. From the respective drop-down menu, select the:
  • Filter
  • Item Master Hierarchy
  • Site Hierarchy
  1. Click Execute. The results for the selected parameters display.

You can control only the active policy settings through the Policy Pivot screen. All changes made on the Policy Pivot screen are considered a user override and, thus, are not overridden by policy tasks (similar to inventory level overrides in the Inventory Manager).

Parameters

The parameters used to display information on the Pivot grid will all be used to narrow the scope of inventories.

Filter: the inventory filter. If Item Master Hierarchy and Site Hierarchy parameters are left blank, all inventories in the filter display in the grid.

Item Master Hierarchy: segments of the product hierarchy may be selected to narrow the scope in the inventory filter selected in the Filter parameter.

Site Hierarchy: segments of the location hierarchy may be selected to narrow the scope in the inventory filter selected in the Filter parameter.

To clear the Item Master Hierarchy and Site Hierarchy parameters, select NULL next to the drop-down menus.

Multiple Changes Tool

The layout of the Pivot grid can be modified using item master and site groupings, as well as all policy settings. Dynamic policies cannot be set up in the Policy Pivot grid, only effective settings.

Changes can be applied in the grid on an individual level, or through the Mass Change tool.

To Change Multiple Data in Columns

  1. Access the Pivot screen. For detail, see To Access the Policy Pivot Screen.
  2. Right-click anywhere on the screen, and select Multiple Changes. The Multi Data Change Form For Column window displays.

  1. Make the necessary changes, and click Save.
