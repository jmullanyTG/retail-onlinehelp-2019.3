Replenishment Strategy

The Replenishment Strategy section on the Update Policy Task screen must be populated with the settings pertinent to the order policy type. It is possible to keep additional settings for inactive types (e.g. bin level for an item that has transitioned from Bin to Normal Stocked).

Safety Stock

Safety stock must be set to Safety Stock Days or Target Service Level. A minimum safety stock units can be set, if needed.

Bin Level

For Bin items, the Bin level can be set up for a specific number of units using Bin Level Units or Target Service Level.
