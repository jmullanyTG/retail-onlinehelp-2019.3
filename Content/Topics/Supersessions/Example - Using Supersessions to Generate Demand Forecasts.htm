Example - Forecasting Supersession

In this example, you will use Demand Management's supersessions feature to create a demand forecast for a new item, High Collar Angelina Jacket, using the sales history of an old item, Leather Moto Jacket. This example shows how to create a supersession and demand forecast for a single color/size combination of the new item. In practice, you will need to repeat these steps for each color/size combination your new items possess.

Instructions

1. Create the Supersession
  1. From the Execute tab, click Supersession. The Superssession Grid displays.
  2. From the toolbar, click New The Items window displays.
  3. From the Filter drop-down menu, select the filter you want to use for the supersession. In this example, select Supersession Model Management (by Dep).
  4. Add the condition condition ItemMaster Group 3 Description is One of the Following: Coats & Jackets.

  5. From the Add Supersession Wizard, select the following old item to supersede:
  • Item Master Code: 543
  • Item Master Description: Leather Moto Jacket_Dark Navy-L

  1. Click Next , and select the following new item to supersede the Leather Moto Jacket_Dark Navy-L:
  • Item Master Code: 708
  • Item Master Description: High Collar Angelina Jacket_Grey - L

  1. Click Next, and select he following supersession parameters:
  • Old Item Replenishment Strategy: No change.
  • Supersession Type: Forecasting only.

  1. Click Next, and select the sites where you want to apply the supersession. In this example, all sites are selected.

  2. Click Next to continue.
  3. Click Finish. The superssession is created.
 2. Generate the Demand Forecast
  1. From the Execute tab, click Inventory Forecast. The Item - Level View window displays.
  2. From the Filter drop-down menu, select Supersession Model Management (by Dep).
  3. Add the condition ItemMaster Group 3 Description is One of the Following: Coats & Jackets.
  4. Select the Supersession Model Management (by Dep) filter.
  5. Add condition: ItemMaster Group 3 Description is One of the Following: Coats & Jackets.
  6. From the Forecasts panel, select the new item High Collar Angelina Jacket.
  7. Create a demand forecast for the new item:
  • In the Forecast toolbar, click Forecast.
  • In the Forecast Task window, click OK to star the forecast tournament.
  1. After the forecast is complete, it displays on the Forecast screen.
