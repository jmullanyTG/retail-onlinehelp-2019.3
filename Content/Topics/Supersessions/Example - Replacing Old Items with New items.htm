Example - Replacing Old Items with New Items

In this example, we use Demand Management's supersessions feature to create a demand forecast for a new item, High Collar Angelina Jacket_Grey, using the sales history of an old item, Leather Moto Jacket_Dark Navy.

To Replace an Old Item with a New Item

  1. From the Execute tab, click Supersessions. The Supersession Grid displays.
  2. From the toolbar, click New The Items window displays.
  3. From the Filter drop-down menu, select the filter you want to use for the supersession. In this example, select the filter 3.2 BC-3 CD-1 SD-1: Supersession Example.
  4. Click OK. The Add Supersession wizard displays.
  5. In the Add Supersession wizard, select the following old item to supersede:
  • Item Master Code: 543
  • Item Master Description: Leather Moto Jacket_Dark Navy-L

  1. Click Next, and select the following new item to supersede the Leather Moto Jacket_Dark Navy-L:
  • Item Master Code: 708
  • Item Master Description High Collar Angelina Jacket_Grey - L

  1. Click Next, and select the following supersession parameters:
  • Old Item Replenishment Strategy: No change
  • Supersession Type: Forecasting

  1. Click Next , and select the sites where you want to apply the supersession. In this example, all sites are selected.

  1. Click Next to continue.
  2. Click Finish to close the wizard and create the supersession.
