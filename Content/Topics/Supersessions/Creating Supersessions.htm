Creating Supersessions

Before you create a new supersession, you must add a filter that includes the new and old items that are supersede .

To Create a new Supersession

  1. From the Execute tab, click Supersession. The Supersession Grid displays.
  2. From the toolbar, click New The Items window displays.
  3. From the Filter drop-down menu, select the filter you want to use for the supersession.

  4. Click OK. The Add Supersession Wizard displays.
  5. In the Add supersession wizard, select the existing items to supersede.

  6. Click Next to continue.

  7. Select the new item that will replace the old item.

  8. Click Next. The Superssession Parameters window displays.

  9. Review or edit the supersession parameters.
  10. Click Next, and select the sites where you want to apply the supersession.

  11. Click Next to continue.
  12. Click Finish. The supersession is created and displays on the Supersession Grid screen
  13. From the Forecast screen, you can now manually generate demand forecasts for the new items, or wait for the forecasts to be generated by the Daily Job Manager.
