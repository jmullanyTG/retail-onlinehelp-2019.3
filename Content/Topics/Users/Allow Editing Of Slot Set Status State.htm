Allow Editing Of Slot Set Status State

Provides access to modify the Slot Set Status State. When all Slot Sets have a Slot Set Status State of Complete, it will trigger a Workflow transition.

This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States

Modifying the Slot Set Status State may modify your access depending on the Event Workflow Slot Set Status State Blacklist configuration

The User Right “Allow Editing of Slot Set Header” must also be granted

This functionality and its placement on the screen is the same whether accessed from the List or Layout
