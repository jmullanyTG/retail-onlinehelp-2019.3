Allow Editing Of Event

Provides access to view and edit an Event including name, Campaign, Event Type, Ad Dates, Sale Dates, Event Budget, Event Type, Tags, description, Geographies, Budget, Event Compositions, Reports, Slot Sets, Slots and Status.

This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States

Without this permission, Event information cannot be viewed
