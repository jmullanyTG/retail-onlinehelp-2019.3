Config - Allow Editing Of Price Display Format

Price Display Formats are created automatically when a new Promotion Strategy is configured. For more information, see “Promotion Strategies” in Promotions Management Standard Documentation. As such, Price Display Formats cannot be created or deleted manually. This User Right provides access to edit a Price Display Format including its dimensions, and placement of text and Promotion fields.

This User Right should only be granted to trusted Super Users
