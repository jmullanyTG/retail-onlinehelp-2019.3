Workload Queue User Rights

Access to functionality in Workload Queue depends on user rights. To

To set up user rights and roles, see Managing User Rights and Roles.

E Description

Allow Access to Workload Queue Module

Enables the Workload Queue icon on the Maintain tab.

Allow Job Reprioritization and Removal

Change a batch’s prioritization and removal from the Active list to be processed.

Allow Workload Queue Configuration

Enables the Configurationbutton on the Workload Queue toolbar, which allows users to configure threshold overrides and optimizations.
