Allow Deleting Of Slot Set

Provides access to delete a Slot Set from an Event. All Promotions that were placed on the Slot Set will be removed but not deleted. Event specific Content associated with those Promotions when they were on the Slot Set will be deleted.

This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States

This functionality and its placement on the screen is the same whether accessed from the List or Layout
